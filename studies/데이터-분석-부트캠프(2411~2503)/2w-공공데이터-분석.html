📊 Seaborn

Seaborn은 Python의 데이터 시각화 라이브러리로, 통계적 그래프를 쉽게 그릴 수 있습니다. Matplotlib을 기반으로 만들어졌으며, 고급 그래프를 간단한 코드로 생성할 수 있습니다.

📝 Seaborn의 주요 특징

  1. 간편한 시각화: 복잡한 통계 그래프를 간단한 코드로 생성 가능.
  2. 통계적 그래프: 히스토그램, 상관관계 히트맵, 박스 플롯 등 통계 시각화 지원.
  3. 다양한 테마 및 스타일: 그래프의 스타일을 쉽게 설정 가능.
  4. Pandas 데이터프레임과의 호환성: 데이터프레임을 바로 사용할 수 있음.

📊 Seaborn 핵심 기능

1. 히스토그램 (Histogram)

  • 데이터 분포를 확인할 때 사용합니다.
import seaborn as sns
import matplotlib.pyplot as plt

sns.histplot(data=df, x='column_name', bins=30, kde=True)
plt.show()

2. 산점도 (Scatter Plot)

  • 두 변수 간의 관계를 시각화합니다.
sns.scatterplot(data=df, x='column1', y='column2', hue='category')
plt.show()

3. 상관관계 히트맵 (Correlation Heatmap)

  • 변수 간 상관관계를 색상으로 나타냅니다.
sns.heatmap(df.corr(), annot=True, cmap='coolwarm')
plt.show()

4. 박스 플롯 (Box Plot)

  • 데이터의 분포와 이상치를 확인합니다.
sns.boxplot(data=df, x='category', y='value')
plt.show()

5. 페어플롯 (Pair Plot)

  • 여러 변수 간의 관계를 한 번에 시각화합니다.
sns.pairplot(df, hue='category')
plt.show()

6. 선 그래프 (Line Plot)

  • 시계열 데이터나 연속형 데이터의 추세를 시각화합니다.
sns.lineplot(data=df, x='date', y='value')
plt.show()

🗺️ Folium

Folium은 지도 기반 시각화를 위한 Python 라이브러리입니다. Leaflet.js를 기반으로 하며, 인터랙티브한 지도를 생성할 수 있습니다.

📝 Folium의 주요 특징

  1. 지도 시각화: 인터랙티브한 지도 생성 가능.
  2. 다양한 마커 지원: 위치를 표시하는 마커 및 팝업 기능.
  3. 다양한 타일 제공: OpenStreetMap, Stamen, CartoDB 등 여러 타일 사용 가능.
  4. GeoJSON 및 Choropleth 지도: 지역별 데이터 시각화 지원.
  5. 간단한 코드로 지도 생성: Python 코드 몇 줄로 지도 생성 가능.

🗺️ Folium 핵심 기능

1. 기본 지도 생성

  • 기본 지도를 생성합니다.
import folium

m = folium.Map(location=[37.5665, 126.9780], zoom_start=12)  # 서울 위치
m

2. 마커 추가

  • 특정 위치에 마커를 추가합니다.
folium.Marker(
    location=[37.5665, 126.9780],
    popup='Seoul City Hall',
    icon=folium.Icon(icon='info-sign')
).add_to(m)

m

3. 원형 마커 추가 (Circle Marker)

  • 반경을 가진 원형 마커를 추가합니다.
folium.CircleMarker(
    location=[37.5665, 126.9780],
    radius=50,
    color='blue',
    fill=True,
    fill_color='cyan'
).add_to(m)

m

4. Choropleth 지도

  • 지역별 데이터를 색상으로 시각화합니다.
folium.Choropleth(
    geo_data='path/to/geojson/file.geojson',
    data=df,
    columns=['region', 'value'],
    key_on='feature.properties.region',
    fill_color='YlGnBu',
    fill_opacity=0.7,
    line_opacity=0.2
).add_to(m)

m

5. GeoJSON 추가

  • GeoJSON 파일을 지도에 추가합니다.
folium.GeoJson('path/to/geojson/file.geojson').add_to(m)
m

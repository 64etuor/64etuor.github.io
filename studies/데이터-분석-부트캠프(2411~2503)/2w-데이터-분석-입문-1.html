🧠 핵심 개념 : Pandas

1. Indexing & Sorting

데이터를 조회하고 정렬하는 방법입니다.

기능

  • Indexing: 특정 행과 열을 선택합니다.

    • .loc[]: 라벨 기반 선택
    • .iloc[]: 정수 기반 선택

  • Sorting: 데이터를 정렬합니다.

    • .sort_values(): 특정 열을 기준으로 정렬
    • .sort_index(): 인덱스를 기준으로 정렬

예제

import pandas as pd

df = pd.DataFrame({
    "Name": ["Alice", "Bob", "Charlie"],
    "Age": [25, 30, 35],
    "City": ["Seoul", "Busan", "Incheon"]
})

# Indexing
print(df.loc[0])        # 라벨 기반
print(df.iloc[1])       # 정수 기반

# Sorting
df_sorted = df.sort_values(by="Age", ascending=False)
print(df_sorted)

2. Filtering

조건에 맞는 데이터를 추출합니다.

기능

  • Boolean Indexing: 조건을 만족하는 행만 필터링
  • 복합 조건: & (AND), | (OR), ~ (NOT) 연산 사용

예제

# Age가 30 이상인 데이터 필터링
filtered = df[df["Age"] >= 30]

# 복합 조건
filtered_multiple = df[(df["Age"] > 25) & (df["City"] == "Busan")]

print(filtered)
print(filtered_multiple)

3. Dates & Times

날짜 및 시간 데이터를 다루는 기능입니다.

기능

  • 날짜 변환: pd.to_datetime()
  • 날짜 연산: 날짜 간 차이 계산
  • 날짜 속성: .dt로 연, 월, 일 추출

예제

df["Date"] = ["2024-01-01", "2024-06-07", "2024-12-31"]
df["Date"] = pd.to_datetime(df["Date"])

# 날짜에서 연도 추출
df["Year"] = df["Date"].dt.year

print(df)

4. GroupBy

데이터를 그룹화하고 집계하는 기능입니다.

기능

  • 그룹화: .groupby()로 데이터를 그룹화
  • 집계 함수: .sum(), .mean(), .count() 등 사용

예제

df = pd.DataFrame({
    "Category": ["A", "B", "A", "B"],
    "Value": [10, 20, 15, 25]
})

grouped = df.groupby("Category")["Value"].sum()
print(grouped)

5. Combine

여러 데이터프레임을 합치는 기능입니다.

기능

  • Concatenate: pd.concat()로 데이터프레임 연결
  • Merge/Join: .merge()로 특정 조건에 맞게 병합

예제

df1 = pd.DataFrame({"ID": [1, 2], "Name": ["Alice", "Bob"]})
df2 = pd.DataFrame({"ID": [1, 2], "Score": [90, 85]})

combined = pd.merge(df1, df2, on="ID")
print(combined)

6. Pivot

데이터를 재구성하는 기능입니다.

기능

  • Pivot Table: .pivot_table()로 데이터 요약
  • Index, Columns, Values 지정

예제

df = pd.DataFrame({
    "Name": ["Alice", "Bob", "Alice", "Bob"],
    "Subject": ["Math", "Math", "Science", "Science"],
    "Score": [90, 85, 95, 80]
})

pivot = df.pivot_table(index="Name", columns="Subject", values="Score")
print(pivot)

7. Values & NA

데이터 값과 결측치(NA)를 다루는 기능입니다.

기능

  • 결측치 확인: .isna(), .notna()
  • 결측치 처리: .dropna(), .fillna()로 처리

예제

df = pd.DataFrame({
    "Name": ["Alice", "Bob", None],
    "Score": [90, None, 85]
})

# 결측치 확인
print(df.isna())

# 결측치 채우기
df_filled = df.fillna({"Name": "Unknown", "Score": 0})
print(df_filled)

🧠 핵심 개념: Plotly

📊 Plotly란?

Plotly는 인터랙티브한 차트를 만들 수 있는 파이썬 데이터 시각화 라이브러리입니다. 웹 기반의 그래프를 생성하며, 줌, 팬, 호버 등과 같은 상호작용 기능을 제공합니다.

📈 Plotly의 기본 사용법

1. 선 그래프(Line Chart)

import plotly.graph_objects as go

fig = go.Figure(data=go.Scatter(x=[1, 2, 3, 4], y=[10, 11, 12, 13], mode='lines'))
fig.update_layout(title='Line Chart Example')
fig.show()

2. 막대 그래프(Bar Chart)

import plotly.graph_objects as go

fig = go.Figure(data=go.Bar(x=['A', 'B', 'C'], y=[10, 20, 30]))
fig.update_layout(title='Bar Chart Example')
fig.show()

3. 산점도(Scatter Plot)

import plotly.graph_objects as go

fig = go.Figure(data=go.Scatter(x=[1, 2, 3, 4], y=[10, 15, 13, 17], mode='markers'))
fig.update_layout(title='Scatter Plot Example')
fig.show()

📊 Plotly Express

Plotly Express는 Plotly의 간단한 인터페이스로, 한 줄로 다양한 차트를 만들 수 있습니다.

예제

import plotly.express as px

# 선 그래프
fig = px.line(x=[1, 2, 3], y=[3, 1, 6], title='Line Chart with Plotly Express')
fig.show()

# 막대 그래프
fig = px.bar(x=['A', 'B', 'C'], y=[4, 7, 2], title='Bar Chart with Plotly Express')
fig.show()

🔄 주요 차트 유형

차트 유형 설명 함수
선 그래프 데이터의 흐름을 시각화 go.Scatter(mode='lines')
막대 그래프 카테고리별 데이터 비교 go.Bar()
산점도 두 변수 간의 관계 확인 go.Scatter(mode='markers')
히스토그램 데이터 분포 확인 go.Histogram()
원형 그래프 비율을 시각화 go.Pie()
박스 플롯 데이터의 분포와 이상치 확인 go.Box()
버블 차트 산점도에 크기 추가 go.Scatter(marker=dict(size))
3D 그래프 3차원 데이터 시각화 go.Scatter3d()

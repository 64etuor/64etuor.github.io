2. 빅쿼리 활용

핵심 내용:

  1. BigQuery 연동

    from google.oauth2 import service_account
from google.cloud import bigquery

SERVICE_ACCOUNT_FILE = "./api_key.json"
credentials = service_account.Credentials.from_service_account_file(SERVICE_ACCOUNT_FILE)
client = bigquery.Client(credentials=credentials, project="project-id")

  2. 쿼리 실행 및 데이터 불러오기

    def import_bigquery_data(query):
    query_job = client.query(query)
    return query_job.to_dataframe()

data = import_bigquery_data("""
    SELECT * FROM `public-data`
  
    LIMIT 10
""")

  3. BigQuery 데이터를 Pandas로 처리

    products = import_bigquery_data("SELECT * FROM `public-data.table`")

    • 매출 데이터 전처리 및 병합

})

ratio = (count_data / count_data.sum()).cumsum()

  2. 파레토 차트 시각화

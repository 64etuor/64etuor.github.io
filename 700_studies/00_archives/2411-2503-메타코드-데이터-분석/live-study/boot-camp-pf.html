boot camp pf

금융 - 김강산 님

현상 → 문제 도출 → 가설(파생변수 설정) → 파생변수에 따른 데이터 분석 → 유효한 파생 변수 확인 → 상황에 맞는 맞는 분석 모델 결정 → F1 score → 주요 변수 Importance 측정 → SHAP value 평가 → 결론 및 시사

금융 - 오다민 님

배경 → 데이터 소개 → 데이터 탐색 → 파생변수 → 모델 설계 → 모델 정교화 → 결론

프로덕트 - 이현정 님

프로젝트 주제 소개 → 개선 과제 소개 → 데이터 탐색 → 과제 개선을 위한 세부 데이터 분석 → 인사이트 → 액션 플랜

프로덕트 - 김송하 님

프로젝트 주제 소개 → 데이터 탐색 → 개선점 도출 → 고객 Segmentation → 매출 향상 전략 도출 → 액션 아이템 도출 → 향후 계획

게임 - ?? 님

포코포코 게임 소개 → 분석 대상 소개 → 유저 콘텐츠 소비 분석 → 유저 플레이 성향 분석 → 재화 분석 → 재화 획득과 사용에 퍼널 분석 진행 → 개선 전략 도출
처음 보는 주제인데도 의사 전달이 명확했고 시각화와 스토리가 잘 이어졌다. 다만 데이터 활용이 어떻게 이루어진 것인지 분석 과정에 대한 세부 설명이 없어서 조금 아쉽

게임 - 노아현 님

  • 사이퍼즈 게임 소개 → 분석 주제에 대한 소개(유입 경로 유효서 확인) → 신규 유저의 유입 분석 → 신규 캐릭터 출시와 유입의 상관 관계 분석 → 스트리머 협업 이벤트와 유입의 상관 관계 분석, ROI 분석 → 마케팅 캠페인 소개 → 마케팅 캠페인 효과 분석 → 첫번째 결론
  • 구매 패턴 및 매출 분석 → 사용 데이터 소개 → 비즈니스 모델 소개 → 구매 패턴 및 매출 분석 → 유저 Segmentation(구매 패턴) → 매출 변동 분석 → 채팅 내역 분석(신규 캐릭터에 대한 메세지 감정 분석) → 구매 아이템에 대한 시계열 매출 분석 → 결론
    스토리가 너무 매끄럽게 이어짐. 스크립트가 준비된 거처럼 전환이 아주 자연스럽다. 이미 분석가라고 해도 믿을 듯

이커머스 - 채?최 ?소희님?

분석 배경 및 목적 → 분석 계획 소개 → EDA → Key metric → User 분석 → Category 분석 → RFM 분석 → 인사이트

이커머스 - ?? 님

분석 배경 및 목적 → 분석 계획 소개 → EDA → RFM 분석 → 주요 변수 식별 → 예측 모델링 → 분석 결과 → 인사이트 요약 →

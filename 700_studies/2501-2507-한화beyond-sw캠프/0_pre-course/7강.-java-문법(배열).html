7강. JAVA 문법(배열)

이번 강의에서는 배열 의 개념과 사용법을 자세히 다루고 있습니다. 배열을 활용하면 여러 개의 데이터를 효율적으로 저장하고, 평균 등 다양한 수치 계산을 간편하게 수행할 수 있습니다. 특히, 배열의 인덱스를 통해 각 요소에 쉽게 접근할 수 있는 방법을 알리며, 반복문을 통해 데이터를 누적하거나 처리하는 기술을 설명합니다. 배열은 고정된 길이 를 갖지만, 다양한 데이터 타입을 저장할 수 있어 매우 유용한 데이터 구조임을 강조합니다. 이 영상을 통해 배열의 기본 사용법을 확실히 익힐 수 있습니다.

1. 📊 배열의 개념과 활용

  • 배열은 동일한 타입의 데이터를 연속된 공간에 저장하여 효율적으로 많은 양의 데이터를 처리할 수 있도록 돕는다. 예를 들어, 30명의 성적을 저장하고 평균 값을 구하는 프로그램을 통해 배열의 필요성을 설명할 수 있다.
  • 생성된 배열은 길이가 고정되며, 인덱스를 통해 배열의 요소에 접근할 수 있다. 인덱스는 0부터 시작하여 배열의 크기보다 1 작은 값까지를 사용한다.
  • 반복문을 사용하면 배열의 각 요소에 접근하고 누적하여 평균을 계산할 수 있다. 예를 들어, 초기값을 0으로 설정하고 각 요소를 더하여 30으로 나누면 평균이 구해지는 방식으로 설명된다.

2. 📊 배열의 선언과 초기화 방법

  • 배열을 변수로 선언할 때는 대괄호를 사용해야 하며, 변수명 뒤에 대괄호를 추가해 배열형태를 표기할 수 있다.
  • 배열이 객체가 없는 경우, null 값으로 초기화할 수 있다.
  • 배열을 생성할 때는 값을 중괄호를 통해 바로 입력하거나, new 연산자를 사용해 메모리공간을 할당할 수 있다.
  • 30칸짜리 배열을 생성하면, 메모리가 확보되고, 그 배열의 변수를 선언하면서 동시에 값들을 넣을 수 있다.
  • 예를 들어, INT 타입 배열을 선언할 때 대괄호를 붙이고, 중괄호를 사용해 값을 할당하는 과정이 필요하다.

3. 🔍 배열의 첫 번째 요소 접근 방법

  • 스코어 변수를 사용하여 대괄호 안에 인덱스번호를 입력함으로써 첫 번째 요소에 접근할 수 있다.
  • 해당 인덱스번호에 맞는 요소가 출력된다.
  • 이 과정을 쉽고 빠르게 수행하기 위해, 컨트롤 D 키를 눌러 복사한 후 숫자만 바꾸는 방법으로 여러 요소에 적용할 수 있다.

4. 🖥️ JAVA 배열과 for문 활용

  • INT sum이라는 변수를 초기 값 0으로 설정한 뒤 for문을 사용하여 인덱스0부터 2까지 반복하는 방식으로 총합을 구한다.
  • 총합은 sum 변수에 누적되고, 이를 통해 평균값을 더블 타입으로 계산하여 출력한다.
  • 배열선언 및 초기화를 위해 new 연산자를 사용해야 하며, 배열에 값을 나중에 넣는 경우에도 new를 사용해야 한다.
  • 기본 데이터 타입은 각각의 초기값이 있으며, int는 0, boolean은 false, 참조 타입은 null로 초기화된다.
  • 배열의 길이는 length 속성을 통해 알아낼 수 있으며, 읽기 전용이기 때문에 수정할 수 없다.

4.1. ️ 배열과 반복문을 이용한 누적 합산

  • INT sum 변수를 0으로 초기화하여 누적 합산을 할 준비를 한다.
  • 반복문을 사용하여 인덱스 0부터 2까지 요소에 접근하며 반복적으로 작업을 수행한다.
  • 초기값을 0으로 설정하고, 조건문을 사용하여 인덱스가 3이 되었을 때 반복을 종료하도록 작성한다.
  • 누적된 값은 sum이라는 변수에 저장되며, 이를 통해 스코어 값을 누적하게 된다.
  • 인덱스번호는 변수 i로 매치하여 사용하고, 반복문을 통해 요소에 접근한다.

4.2. 배열을 통한 데이터 처리 결과

  • 총합을 구한 후에 결과를 출력하는 단계가 포함된다.
  • 평균값은 더블 타입으로써, 총합을 3으로 나눈 결과로 계산된다.
  • 실행된 결과에서는 각 요소별로 잘 출력된 값을 확인할 수 있다.
  • 총합이 260이라는 값으로 계산되었으며, 평균값은 약 86.67로 나타났다.
  • 배열을 사용하여 여러 값을 동시에 저장하고 처리할 수 있다는 점을 확인하였다.

4.3. 배열의 생성 및 초기화

  • 예제 2를 통해 'scores'라는 배열변수를 선언하고, 배열의 참조변수를 먼저 선언한 후 값을 넣는 과정을 설명한다.
  • 배열 초기화는 new 키워드를 사용하여 값 목록을 넣어야 하며, 선언과 별개로 실행문을 적성을 할 경우 초기화를 허용하지 않는다.
  • 총합을 구하는 코드에서는 int 배열을 0으로 초기화하고, for문을 이용하여 값을 더해 출력하는 방법을 보여준다.
  • 평균값계산을 위해 double 타입으로 변환 후, 배열의 합을 배열의 크기로 나누어 평균을 구하는 과정을 설명한다.
  • 배열생성 시, 선언 후 나중에 값을 넣을 때는 new 연산자를 사용하여 배열의 길이를 지정하고 값을 나열하는 것을 강조한다.

4.4. 배열 초기화 및 사용

  • 배열 생성 시 기본값이 자동으로 초기화되며, 기본 타입은 각각 특정 값을 가지며, 예를 들어, int와 float는 0, boolean은 false, 객체참조 타입은 null로 초기화된다.
  • 배열을 생성하는 방법은 new 연산자를 사용하며, 크기를 지정하여 사용할 수 있고, 예를 들어 new int[3]는 3개의 요소를 가지고 있는 정수형 배열을 만든다.
  • 각 인덱스에 값 할당 후 출력하는 방법을 통해 배열의 사용 예를 보여주며, 예를 들어 0번째 인덱스에 10, 1번째 인덱스에 20과 같은 방식으로 값을 할당한다.
  • 배열의 길이를 알기 위한 방법으로 length 필드를 사용하여 배열의 크기를 확인할 수 있으며, 이 값은 읽기 전용이라 변경할 수 없다.
  • 프로그램에서 메인 메소드의 매개변수로 문자열을 전달하고, 이를 정수로 변환하여 덧셈 결과를 출력하는 방법도 설명되며, 이 경우 하드코딩의 문제를 해결할 수 있다.

4.5. ️ 다차원 배열의 개념 및 사용법

  • 다차원 배열은 대괄호를 두 차례 연속해서 사용하여 생성을 하며, 이때 첫 번째 대괄호는 행을, 두 번째는 열을 의미한다.
  • 인트 타입의 배열인 abc는 배열안에 배열을 참조하는 구조를 가지며, 각 참조 변수는 배열값을 내포하게 된다.
  • 인덱싱을 통해 0번째 요소에 접근할 경우 배열 b를 참조할 수 있고, 그 안의 값에 대해 인덱스0, 1, 2로 접근이 가능하다.
  • 각 배열의 길이는 length 메소드를 통해 구할 수 있으며, 다차원 배열은 배열안에 서로 다른 길이의 배열을 포함할 수 있는 것이 큰 특징이다.
  • 배열을 중첩해 직접 값을 할당할 수 있으며, 중괄호를 통해 값을 입력하여 접근하는 예시로 95와 80, 92와 96이라는 값을 사용할 수 있다.

4.6. 객체를 참조하는 배열의 이해

  • 객체를 참조하는 참조 타입의 배열을 만들 수 있으며, 이때 배열의 타입은 동일해야 한다.
  • 스트링 배열을 생성할 때, 배열의 크기는 예를 들어 3으로 설정할 수 있고, 각 요소에 값을 할당할 수 있다.
  • 문자열 리터럴 값을 동일하게 할당하면 동일한 객체를 참조하게 되고, new 연산자를 사용하여 객체를 생성하면 서로 다른 메모리공간에서 생성된다.
  • 두 객체가 같은 값을 가지더라도, 서로 다른 메모리위치를 가리키면 값을 비교할 때 서로 다르다고 평가된다.
  • equals 메소드를 통해 내용 비교를 진행할 경우, 값이 동일하다면 true를 반환한다.

4.7. 배열 복사의 방법

  • 배열을 복사할 때는 for문을 사용하거나 Arrays.copyOf 메소드를 활용할 수 있다.
  • 예제를 통해 Exercise 07이라는 이름으로 배열복사를 시도하며, 1, 2, 3 값을 갖는 배열을 생성한다.
  • for문을 사용하여 새로운 배열에 기존 배열의 값을 복사하는 과정을 설명하며, 인덱스를 서로 연동한다.
  • 복사 결과 기존 배열의 값과 길이가 맞지 않아 남은 인덱스는 0으로 유지됨을 확인할 수 있다.
  • Array.copy 메소드를 통해 스트링 배열 복사를 예示하며, 복사할 범위를 설정하여 복사하는 방법을 보여준다.
  • 향상된 for문을 설명하며, 배열의 요소를 반복적으로 꺼내어 처리하는 방법과 종료 조건을 설명한다.

4.8. 배열과 향상된 for문 활용

  • INT 타입의 배열을 생성하고, 원하는 값을 할당하는 작업을 수행한다.
  • 향상된 for문을 사용하여 배열의 요소를 하나씩 꺼내오는 방식을 적용한다.
  • 꺼내온 각 요소는 변수 스코어에 담기며, 반복문 내에서 총합을 구하는 작업이 이루어진다.
  • 더블 타입을 활용하여 평균값을 구하며, 총합을 배열의 길이로 나누는 방식으로 계산한다.
  • 출력 과정에서 평균값을 S 아웃으로 표시하고, 결과를 확인하는 절차를 포함한다.

5. 📚 배열의 기초 이해

  • 배열은 동일한 타입의 값들을 여러 개 가질 수 있는 유용한 자료 구조이다.
  • 배열의 길이는 고정적으로 설정되며, 이후 가변적으로 변할 수 없다.
  • 배열에는 기본 타입의 값과 객체 타입(참조 타입) 모두 저장할 수 있다.
  • for문을 사용하면 배열의 각 요소를 꺼내어 사용할 수 있다.
  • 향상된 for문을 통해 각 요소들을 쉽게 꺼내어 처리할 수 있다.

Key term

JAVA

JAVA는 컴퓨터 프로그램을 만들기 위해 사용하는 프로그래밍 언어입니다. 이 언어는 많은 웹사이트와 앱에서 사용되고 있습니다. JAVA는 '한 번 작성하면 어디서나 실행할 수 있다'는 특징이 있어요.

배열

배열은 같은 종류의 여러 개의 데이터를 저장할 수 있는 특별한 상자입니다. 이 상자는 여러 개의 칸으로 나뉘어져 있어서 각 칸에 데이터를 넣을 수 있습니다. 예를 들어, 학생 30명의 점수를 담을 수 있는 30개의 칸이 있는 상자 같아요.

데이터 처리

데이터 처리는 컴퓨터가 데이터를 수집, 변환 및 분석하는 과정입니다. 데이터를 정리하여 필요한 정보를 찾는 것을 의미하죠. 마치 여러 종이자료를 정리하여 보고서를 만드는 작업과 비슷합니다.

인덱스

인덱스는 배열의 각 칸을 가리키는 번호입니다. 배열의 첫 번째 칸은 0번, 두 번째 칸은 1번처럼 0부터 시작해요. 그래서 인덱스를 사용하면 원하는 데이터를 쉽게 찾을 수 있습니다.

값 누적

값 누적은 여러 개의 값을 더하여 하나의 결과를 만드는 과정입니다. 예를 들어, 학생들의 점수를 모두 더한 후 평균을 구하는 것이죠. 여러 점수를 하나로 합치는 작업이라고 생각하면 돼요.

for문

for문은 반복적인 작업을 쉽게 할 수 있게 해주는 문장입니다. 특정 조건에 맞을 때까지 같은 일을 반복할 수 있어요. 예를 들어, 학생 30명의 점수를 하나씩 읽는 것을 간단하게 반복하죠.

평균값

평균값은 여러 개의 숫자의 합을 개수로 나눈 결과입니다. 예를 들어, 세 명의 학생 점수 80, 90, 85가 있다면 평균값은 (80 + 90 + 85) / 3 = 85가 됩니다.

형 변환

형 변환은 데이터의 종류를 바꿔주는 과정입니다. 예를 들어, 정수를 실수로 바꾸는 것이죠. 이는 계산할 때 필요한 경우 사용할 수 있어요.

참조 변수

참조 변수는 다른 데이터의 위치를 가리키는 변수입니다. 즉, 어떤 위치에 저장된 데이터를 가리키는 주소를 가지고 있습니다. 집 주소를 알고 있는 것과 비슷하죠.

및 참조 타입

참조 타입은 참조 변수를 사용하는 데이터 형식입니다. 배열이나 클래스와 같이 여러 값을 그룹으로 다룰 수 있어요. 기본 데이터와 반대로 '무엇'을 담아 반환하는 형식입니다.

2차원 배열

2차원 배열은 각각의 칸이 다시 배열을 가지는 구조입니다. 예를 들어, 행렬처럼 가로와 세로로 배열된 형태죠. 수학에서 행렬을 다루는 것과 비슷합니다.

다차원 배열

다차원 배열은 여러 개의 방향으로 데이터를 저장하는 배열입니다. 예를 들어, 2차원 배열은 가로와 세로에 데이터를 저장하죠. 복잡한 데이터 구조를 표현할 수 있는 도구입니다.

기본 타입

기본 타입은 정수, 실수, 불리언 등과 같은 간단한 데이터 형식입니다. 이를 통해 숫자나 참/거짓 값 등을 비교하고 사용할 수 있습니다. 마치 기초적인 재료와 같아요.

문법

문법은 프로그래밍 언어를 사용할 때 규칙과 형태입니다. 올바른 문법을 써야 오류 없이 코드를 작성할 수 있죠. 언어의 문장과 구조를 정의하는 것입니다.

데이터 구조

데이터 구조는 데이터를 저장하고 관리하는 방법입니다. 배열, 리스트, 필드 등이 특징적이에요. 이는 정보를 효율적으로 저장하는 방식을 말합니다.

배열 생성

배열 생성은 새로운 배열을 만드는 과정입니다. 배열을 사용하려면 먼저 만들어야 하며, 이때 배열의 크기와 타입을 지정합니다. 상자를 새로 만드는 것과 같아요.

초기화

초기화는 변수가 처음 사용되기 전에 기본값을 설정하는 과정입니다. 배열의 모든 칸을 처음 상태로 만들어주죠. 그러니까 상자를 정리하는 작업이라고 볼 수 있어요.

영역

영역은 메모리에서 데이터를 저장하는 공간을 의미합니다. 배열을 생성하면 메모리의 특정 영역에 데이터가 저장됩니다. 데이터를 담는 장소와 같다고 생각하면 됩니다.

메모리

메모리는 컴퓨터가 데이터를 잠시 저장하는 공간입니다. 배열을 만들면 메모리 안에 공간을 할당받아 데이터를 저장하죠. 마치 책상 위에 서류를 놓아두는 것과 비슷해요.

값 할당

값 할당은 변수가 가리키는 메모리 위치에 특정 값을 넣는 과정입니다. 예를 들어, 배열의 0번 칸에 '100'이라는 숫자를 넣는 것이죠. 변수를 설정하는 것과 같아요.

개수

개수는 배열에 들어있는 데이터의 수를 의미합니다. 예를 들어, 학생 성적을 담은 배열에 30명이 있다면 개수는 30입니다. 얼마나 많은 데이터를 가지고 있는지를 나타내죠.

향상된 for문

향상된 for문은 반복문을 쉽게 쓸 수 있게 해주는 특별한 문법입니다. 리스트의 모든 요소를 차례대로 가져올 수 있어요. 많은 데이터를 쉽게 처리하는 방법이라고 생각하면 돼요.

하드코딩

하드코딩은 프로그램에서 숫자나 문자열을 직접 입력하는 것을 의미합니다. 예를 들어, 점수를 30으로 잘라서 고정하면 하드코딩이 됩니다. 프로그램이 변화를 수용하지 못하게 하는 방식이죠.

프로그램

프로그램은 컴퓨터가 특정 작업을 수행하기 위해 작성된 명령의 집합입니다. 여러 과정을 자동으로 처리해줍니다. 일종의 계획서 같은 형태의 작업이라고 할 수 있어요.

에러 처리

에러 처리는 프로그램 실행 과정에서 잘못된 부분을 해결하는 과정입니다. 문제가 발생했을 때 어떻게 대처할지 정해주는 것이죠. 오류를 피하는 방법이라고 할 수 있어요.

기본값

기본값은 변수를 사용하기 전에 자동으로 설정되는 값입니다. 예를 들어, 정수 배열은 처음에 모두 0으로 초기화됩니다. 사용자가 따로 설정하지 않아도 생기는 값이라고 볼 수 있어요.

클래스

클래스는 객체 지향 프로그래밍의 기본 개념으로, 속성과 행동을 정의하는 설계도 역할을 합니다. 여러 객체들을 만들기 위한 틀로 생각하면 됩니다.

객체

객체는 클래스의 실제 인스턴스입니다. 클래스에서 정의된 속성과 행동을 가진 구체적인 데이터 단위를 의미해요. 예를 들어, '자동차'라는 클래스에서 '아반떼'라는 객체를 만들 수 있죠.

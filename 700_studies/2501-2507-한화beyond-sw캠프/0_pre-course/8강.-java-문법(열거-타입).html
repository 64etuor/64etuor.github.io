8강. JAVA 문법(열거 타입)

이번 영상에서는 열거 타입에 대해 자세히 설명합니다. 열거 타입은 한정된 값을 갖는 카테고리화된 데이터로, 코드 구현 시 간편함과 안정성을 제공합니다. 이 강의를 통해 열거 타입을 정의하고 사용하는 방법을 배우고, 메모리에서의 저장 방식과 예제 코드를 통해 이해를 돕습니다. 특히, 스위치 문을 활용한 요일 판별 방법을 학습하여 실용적인 프로그래밍 기술을 익힐 수 있습니다. 열거 타입에 대한 이해는 자바프로그래밍에서 필수적인 요소이므로 초보자들에게 큰 도움이 될 것입니다.

1. 🔍 열거 타입의 기본 개념과 정의

  • 열거 타입은 카테고리화된 데이터들을 나타낸다.
  • 열거 타입은 값들을 한정된 값만 갖게 하기 위해 만들어진 타입이다.
  • 열거 상수를 저장하는 타입을 의미한다.
  • "한정된 값"은 결국 카테고리화된 값들이다.

2. 📋 열거 타입의 기본 이해

  • '너미'라는 새로운 개념이 있으며, 이는 보통 클래스와는 다른 방식으로 정의된다.
  • 열거 상수는 대문자로 작성되며, 이 값들은 콤마로 구분되어 여러 개의 값을 가질 수 있다.
  • 열거 타입을 선언할 때, 코드 블록을 작성하여 필요한 열거 상수를 정의해야 한다.
  • 열거 타입의 변수는 생성 시 열거 타입의 이름과 그에 해당하는 상수를 점으로 구분하여 적어야 한다.
  • 메모리에서는 열거 타입의 객체가 생성되어 그 안에 정의된 값들이 저장되며, 예를 들어 '위기'라는 이너 타입 안에 먼데이에서 선데이까지의 값이 포함된다.

3. 🗂️ 열거 타입의 변수 생성 및 상수 정의

  • 열거 타입의 변수를 만들 때, 해당 이름을 적고 점을 찍어서 열거 상수를 정의해야 한다.
  • 변수가 담고 있는 값이 특정 위계와 일치하는지를 비교하여 확인할 수 있다.
  • 예제를 통해 section 3번을 만들고, exam01이라는 이름으로 변수를 생성하는 과정을 소개한다.
  • 선택 항목 중에서 원하는 항목을 선택하고 엔터를 눌러 생성할 수 있다.
  • 열거 타입의 상수를 정의할 때는 월요일부터 일요일까지의 값을 포함해야 하며, 이를 통해 완성된 열거 타입을 보여준다.

4. 📅 요일 판별 스위치문 구현

  • 스위치문을 사용하여 자바에서 오늘의 요일을 판별하는 예제를 구현한다.
  • 캘린더 객체를 활용하여 요일을 구하기 위해 get(Calendar.DAY_OF_WEEK) 메서드를 사용한다.
  • 요일에 따라 각각의 케이스 문을 설정하여, 1은 일요일, 2는 월요일 등으로 정의한다.
  • 최종적으로 출력을 통해 오늘의 요일을 확인하며, 그에 따른 다른 코드를 실행하는 로직이 포함된다.
  • 예를 들어, 화요일일 경우 "오늘은 화요일이다"라는 출력이 이루어진다.

4.1. ‍ 열거 타입 사용 예제

  • 타입에 대해 설명하면서 열거 타입 사용 예제를 들어보도록 한다.
  • 예제로 사용할 타입명은 exam 02번으로 설정하고, 그 안에는 enum 및 example이 포함된다.
  • 변수명을 today로 설정하며, 초기값은 null로 설정한다.
  • 캘린더 객체를 사용하여 자바의 클래스중 하나를 활용하는 예제를 다룬다.
  • 현재 캘린더 객체는 자주 사용되지 않으나, 이번 예제에서는 사용될 예정이다.

4.2. 요일을 구분하는 스위치문

  • 캘린더 객체에서 요일 값을 INT 값으로 전달받아 사용한다.
  • 스위치문을 통해 INT 값에 따라 요일을 분류하는 기능을 구현한다.
  • INT 값이 1일 경우 일요일, 2일 경우 월요일, 3일 경우 화요일로 설정된다.
  • INT 값이 4는 수요일, 5는 목요일, 6은 금요일, 7은 토요일로 할당된다.
  • 이러한 분기는 오늘의 요일을 INT 값으로 얻은 후 진행된다.

4.3. ️ 요일 출력 예제

  • sout을 사용하여 오늘의 요일을 출력하는 예제를 소개한다.
  • "오늘 요일은 그래서 투데이다"라는 메시지가 출력된다.
  • "today가 만약에 sunday가 아니라면..."이라는 조건문을 포함해 다른 메시지를 출력할 수 있다.
  • 일요일에는 "축구를 합니다."라는 메시지가 출력되는 방식으로 요일별 행동을 정의할 수 있다.
  • 실행 결과, 오늘이 화요일인 경우 "Tuesday"라는 요일이 출력된다.

5. 🛠️ 열거 타입의 소개와 활용

  • 카테고리성 데이터를 갖는 자료형을 만들어 보았다.
  • 클래스대신에 이러한 형태로 작성을 할 수 있었다.
  • 변수를 담고, 비교하고, 활용하는 방법에 대한 예제를 간략히 설명하였다.

Key term

열거 타입

열거 타입은 한정된 값을 가진 데이터 셋으로, 이를 통해 소프트웨어에서 여러 가지 선택지를 쉽게 관리할 수 있어요. 예를 들어, 요일을 나타낼 때 월요일, 화요일 같은 값들만 사용하도록 정해둘 수 있습니다.

카테고리화

카테고리화란, 비슷한 특성을 가진 것들을 그룹으로 묶는 과정이에요. 프로그래밍에서는 데이터를 더 쉽게 다루기 위해 이처럼 그룹화하여 관리합니다.

열거 상수

열거 상수는 열거 타입 안에서 사용할 수 있는 정해진 값들입니다. 예를 들어, 요일을 표현하는 열거 타입 안에서는 MONDAY, TUESDAY, WEDNESDAY 같은 값이 여기에 해당하죠.

변수 선언

변수 선언은 프로그래밍에서 데이터를 저장하기 위한 이름을 만드는 것이에요. 이는 마치 빈 상자를 만들고 그 상자에 이름을 붙이는 것과 같아요. 이렇게 하면 필요할 때 그 상자에 담긴 값을 쉽게 찾을 수 있습니다.

메모리 영역

메모리 영역은 프로그램이 실행될 때 데이터를 저장하는 공간이에요. 데이터를 저장하려면 개별적인 영역이 필요하듯이, 메모리도 여러 부분으로 나누어져 각각의 데이터가 저장됩니다.

클래스

클래스는 객체 지향 프로그래밍에서 특정한 속성과 기능을 가진 데이터 구조를 정의하는 틀이에요. 이는 마치 설계도와 같아서, 여러 객체를 만들기 위한 기본 틀로 사용됩니다.

소스파일

소스파일은 프로그래머가 작성한 코드가 담긴 파일로, 프로그램의 내용을 담고 있어요. 이는 마치 책의 페이지처럼, 특정한 내용을 기록하는 곳입니다.

비교 연산

비교 연산은 두 개의 값을 비교하여 같거나 다름을 판단하는 작업이에요. 마치 두 사람이 가지고 있는 사탕의 수를 비교하여 누가 더 많은지를 알아보는 것과 같아요.

캘린더 객체

캘린더 객체는 날짜와 시간을 다루기 위한 자바의 클래스로 특정한 방법으로 날짜를 표현하고 관리할 수 있게 해줘요. 예를 들어, 오늘의 날짜를 자동으로 알 수 있게 해줍니다.

스위치 문

스위치 문은 여러 경우의 수 중 하나에 따라 코드를 분기하도록 도와주는 구조이에요. 이는 마치 크리스마스 선물 중에서 어떤 것을 선택할지를 결정하는 것과 같아요.

INT 값

INT 값은 정수형 데이터를 나타내는 데이터 타입의 하나로, 숫자 값을 저장하는 데 사용됩니다. 1, 2, 3 같은 수를 처리할 수 있죠.

데이 오브 위기

데이 오브 위기는 캘린더 객체를 통해 요일을 나타내는 정수 값으로, 요일을 1부터 7까지의 숫자로 표현해요. 예를 들어, 1은 월요일을 의미하죠.

자바

자바는 프로그래밍 언어의 하나로, 다양한 용도로 사용되는 아주 인기 있는 언어입니다. 이 언어를 사용하여 웹사이트나 앱을 만들 수 있어요.

프로그래밍 기술

프로그래밍 기술은 컴퓨터 프로그램을 만들기 위한 기술이나 방법론을 의미해요. 이러한 기술을 통해 우리는 다양한 소프트웨어를 개발할 수 있습니다.

참조 변수

참조 변수는 데이터를 직접 저장하는 것이 아니라 데이터가 저장된 위치를 가리키는 변수에요. 마치 지도에서 집이 있는 위치를 가리키는 것처럼, 실제 데이터는 다른 곳에 있습니다.

값 비교

값 비교는 두 개의 값을 확인하여 같은지 다른지를 판단하는 작업입니다. 이 작업을 통해 프로그램의 흐름이 달라질 수 있어요.

사용 예제

사용 예제는 프로그래밍에서 어떤 기능이나 방법을 실제로 보여주는 코드를 의미합니다. 이를 통해 우리는 어떻게 사용해야 하는지 쉽게 이해할 수 있습니다.

코드 블록

코드 블록은 여러 코드가 하나의 그룹으로 묶인 부분을 말해요. 이는 마치 여러 단어로 문장을 만들듯이, 프로그래밍에서도 여러 코드를 함께 묶어 처리할 수 있습니다.

상수

상수는 프로그램에서 절대 변하지 않는 값을 말해요. 예를 들어, 연필의 색상처럼, 한 번 정해지면 바뀌지 않는 값을 의미합니다.

퍼블릭

퍼블릭은 프로그래밍에서 어디서나 접근할 수 있도록 설정된 속성이에요. 이는 마치 공원 같은 곳으로, 누구나 들어올 수 있는 개방된 장소입니다.

객체

객체는 클래스의 설계도를 바탕으로 만들어진 구체적인 존재로, 실제로 사용되는 데이터와 기능을 포함하고 있어요. 마치 자동차라는 설계도가 있을 때, 실제 자동차가 그 객체입니다.

데이터 유형

데이터 유형은 프로그래밍에서 각 변수에 어떤 종류의 데이터를 저장할 수 있는지를 정의하는 것이에요. 이는 숫자, 문자, 날짜 등 여러 종류가 있을 수 있습니다.

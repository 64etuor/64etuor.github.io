Low Code Tools list

노코드 및 관련 툴 공식 홈페이지 모음

이 문서는 모바일 앱 개발, 웹 개발, 데이터베이스, 자동화, 생산성 도구 등 다양한 노코드 플랫폼을 한눈에 살펴볼 수 있도록 구성되었습니다. 각 항목별 링크와 주요 기능을 상세히 담고 있으니, 필요에 맞는 툴을 직접 확인해보세요.

📱 Mobile App Development

Title URL Feature (특징)
Thunkable https://thunkable.com 드래그 앤 드롭 방식으로 iOS 및 Android 앱을 제작할 수 있는 노코드 플랫폼.
MIT App Inventor https://appinventor.mit.edu 초보자도 쉽게 블록 기반 인터페이스로 모바일 앱을 개발할 수 있는 도구.
AppSheet https://www.appsheet.com Google의 노코드 플랫폼으로, 데이터 기반 모바일 앱 개발을 지원.
Draftbit https://draftbit.com React Native 기반으로 앱을 시각적으로 설계하고 개발할 수 있는 빌더.
Adalo https://www.adalo.com 데이터베이스 통합이 쉬우며, 컴포넌트를 활용한 앱 개발 도구.
Appgyver https://www.appgyver.com 프로 수준의 노코드 앱 개발 플랫폼으로 강력한 백엔드 기능을 제공.
Apprat.io https://apprat.io 빠르게 프로토타이핑이 가능한 앱 빌더로, 디자인 및 기능을 쉽게 구현 가능.
Fliplet https://fliplet.com 엔터프라이즈급 앱 개발에 적합하며 다양한 템플릿 제공.
Glide https://www.glideapps.com 스프레드시트 데이터를 기반으로 빠르게 모바일 및 웹 앱을 제작할 수 있는 도구.

파이썬 내장 함수 목록

파이썬 버전: 3.9

점프 투 파이썬 내장함수 link

내장 함수 목록

함수명 정식 명칭 (English) 설명 문법
abs() Absolute Value 주어진 숫자의 절대값을 반환합니다. abs(x)
all() All Elements True 반복 가능한 객체의 모든 요소가 참이면 True, 아니면 False를 반환합니다. all(iterable)
any() Any Element True 반복 가능한 객체의 적어도 하나의 요소가 참이면 True, 모두 거짓이면 False를 반환합니다. any(iterable)
ascii() ASCII Representation 객체의 ASCII 표현을 포함하는 문자열을 반환합니다. ascii(object)
bin() Binary Representation 정수를 이진수 문자열로 변환합니다. bin(x)
bool() Boolean Conversion 주어진 값을 불린 값(True 또는 False)으로 변환합니다. bool(x)
bytearray() Bytearray Object 주어진 값으로부터 가변 바이트 배열 객체를 생성합니다. bytearray([source[, encoding[, errors]]])
bytes() Bytes Object 주어진 값으로부터 불변 바이트 객체를 생성합니다. bytes([source[, encoding[, errors]]])
callable() Callable Check 객체가 호출 가능한지 여부를 확인합니다. callable(object)
chr() Character from Unicode 유니코드 코드 포인트로부터 해당 문자를 반환합니다. chr(i)
classmethod() Class Method 함수를 클래스 메소드로 변환합니다. classmethod(function)
compile() Compile Code 소스 코드를 코드 객체로 컴파일합니다. compile(source, filename, mode, flags=0, dont_inherit=False, optimize=-1)
complex() Complex Number 복소수를 생성합니다. complex([real[, imag]])
delattr() Delete Attribute 객체의 속성을 삭제합니다. delattr(object, name)
dict() Dictionary Object 새로운 사전 객체를 생성합니다. dict(**kwarg)
dir() Directory Listing 객체의 속성과 메소드 목록을 나열합니다. dir([object])
divmod() Division and Modulus 두 숫자를 나누고, 그 몫과 나머지를 튜플로 반환합니다. divmod(a, b)
enumerate() Enumerate Object 반복 가능한 객체를 인덱스를 포함하는 열거 객체로 변환합니다. enumerate(iterable, start=0)
eval() Evaluate Expression 주어진 표현식을 실행하고 결과를 반환합니다. eval(expression[, globals[, locals]])
exec() Execute Code 동적으로 작성된 파이썬 코드를 실행합니다. exec(object[, globals[, locals]])
filter() Filter Elements 함수와 반복 가능한 객체를 사용하여 필터링된 요소들을 반환합니다. filter(function, iterable)
float() Float Conversion 주어진 값을 실수로 변환합니다. float([x])
format() Format String 주어진 값의 형식 지정 문자열을 반환합니다. format(value[, format_spec])
frozenset() Frozen Set 새로운 불변 집합 객체를 생성합니다. frozenset([iterable])
getattr() Get Attribute 객체의 속성 값을 반환합니다. getattr(object, name[, default])
globals() Global Variables 현재 전역 심볼 테이블을 사전으로 반환합니다. globals()
hasattr() Has Attribute 객체가 해당 속성을 가지고 있는지 확인합니다. hasattr(object, name)
hash() Hash Value 객체의 해시 값을 반환합니다. hash(object)
help() Help System 내장 도움말 시스템을 호출합니다. help([object])
hex() Hexadecimal Representation 정수를 16진수 문자열로 변환합니다. hex(x)
id() Identity 객체의 고유 ID를 반환합니다. id(object)
input() Input 사용자로부터 입력을 받습니다. input([prompt])
int() Integer Conversion 주어진 값을 정수로 변환합니다. int([x])
isinstance() Instance Check 객체가 클래스의 인스턴스인지 확인합니다. isinstance(object, classinfo)
issubclass() Subclass Check 클래스가 다른 클래스의 하위 클래스인지 확인합니다. issubclass(class, classinfo)
iter() Iterator 반복 가능한 객체의 반복자를 반환합니다. iter(object[, sentinel])
len() Length 객체의 길이(요소의 수)를 반환합니다. len(s)
list() List Object 새로운 리스트 객체를 생성합니다. list([iterable])
locals() Local Variables 현재 지역 심볼 테이블을 사전으로 반환합니다. locals()
map() Map Function 함수와 반복 가능한 객체를 사용하여 매핑된 요소들을 반환합니다. map(function, iterable, ...)
max() Maximum Value 반복 가능한 객체의 최대값을 반환합니다. max(iterable, *[, key, default])
memoryview() Memory View 객체의 메모리 뷰를 반환합니다. memoryview(obj)
min() Minimum Value 반복 가능한 객체의 최소값을 반환합니다. min(iterable, *[, key, default])
next() Next Element 반복자의 다음 요소를 반환합니다. next(iterator[, default])
object() Object 새로운 빈 객체를 생성합니다. object()
oct() Octal Representation 정수를 8진수 문자열로 변환합니다. oct(x)
open() Open File 파일을 열고 파일 객체를 반환합니다. open(file, mode='r', buffering=-1, encoding=None, errors=None, newline=None, closefd=True, opener=None)
ord() Unicode Value 문자의 유니코드 코드 포인트를 반환합니다. ord(c)
pow() Power Calculation 주어진 값의 제곱을 계산하여 반환합니다. pow(x, y[, z])
print() Print 주어진 값을 출력합니다. print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
property() Property 속성을 정의하는 새로운 속성 객체를 반환합니다. property(fget=None, fset=None, fdel=None, doc=None)
range() Range Object 주어진 범위의 정수 시퀀스를 생성하는 범위 객체를 반환합니다. range(stop) or range(start, stop[, step])
repr() Representation 객체의 공식 문자열 표현을 반환합니다. repr(object)
reversed() Reversed Iterator 시퀀스의 역순 반복자를 반환합니다. reversed(seq)
round() Round Number 주어진 숫자를 반올림하여 반환합니다. round(number[, ndigits])
set() Set Object 새로운 집합 객체를 생성합니다. set([iterable])
setattr() Set Attribute 객체의 속성 값을 설정합니다. setattr(object, name, value)
slice() Slice Object 주어진 인덱스를 사용하여 슬라이스 객체를 생성합니다. slice(stop) or slice(start, stop[, step])
sorted() Sorted List 반복 가능한 객체를 정렬하여 새로운 리스트로 반환합니다. sorted(iterable, *, key=None, reverse=False)
staticmethod() Static Method 함수를 정적 메소드로 변환합니다. staticmethod(function)
str() String Conversion 주어진 값을 문자열로 변환합니다. str(object='') or str(object=b'', encoding='utf-8', errors='strict')
sum() Sum of Elements 반복 가능한 객체의 합계를 계산하여 반환합니다. sum(iterable, /, start=0)
super() Super Object 부모 클래스의 메소드를 호출합니다. `super([type

파이썬 내장 오류 목록

파이썬 버전: 3.9

오류명 정식 명칭 (English) 설명 문법
BaseException Base Exception 모든 예외의 기본 클래스입니다. 직접적으로 사용되기보다는 상속을 통해 새로운 예외 클래스를 정의하는데 사용됩니다. raise BaseException("error message")
Exception Exception 모든 비시스템 예외의 기본 클래스입니다. raise Exception("error message")
ArithmeticError Arithmetic Error 모든 산술 오류의 기본 클래스입니다. raise ArithmeticError("error message")
BufferError Buffer Error 버퍼 관련 작업에서 발생하는 오류입니다. raise BufferError("error message")
LookupError Lookup Error 시퀀스와 매핑 관련 조회 오류의 기본 클래스입니다. raise LookupError("error message")
AssertionError Assertion Error assert 문이 실패할 때 발생합니다. raise AssertionError("error message")
AttributeError Attribute Error 속성 참조 또는 할당이 실패할 때 발생합니다. raise AttributeError("error message")
EOFError End Of File Error input() 함수가 EOF를 만나면 발생합니다. raise EOFError("error message")
FloatingPointError Floating Point Error 부동 소수점 연산에서 오류가 발생할 때 발생합니다. raise FloatingPointError("error message")
GeneratorExit Generator Exit 제네레이터가 close() 메서드에 의해 종료될 때 발생합니다. raise GeneratorExit("error message")
ImportError Import Error import 문이 모듈을 찾지 못할 때 발생합니다. raise ImportError("error message")
ModuleNotFoundError Module Not Found Error import 문이 특정 모듈을 찾지 못할 때 발생합니다. raise ModuleNotFoundError("error message")
IndexError Index Error 시퀀스 인덱스가 범위를 벗어날 때 발생합니다. raise IndexError("error message")
KeyError Key Error 딕셔너리에서 키가 존재하지 않을 때 발생합니다. raise KeyError("error message")
KeyboardInterrupt Keyboard Interrupt 사용자가 인터럽트 키(Ctrl+C 또는 Delete)를 누를 때 발생합니다. raise KeyboardInterrupt("error message")
MemoryError Memory Error 메모리가 부족할 때 발생합니다. raise MemoryError("error message")
NameError Name Error 지역 또는 전역 변수가 찾을 수 없을 때 발생합니다. raise NameError("error message")
NotImplementedError Not Implemented Error 추상 메서드가 구현되지 않았을 때 발생합니다. raise NotImplementedError("error message")
OSError OS Error 운영 체제에서 발생하는 오류입니다. raise OSError("error message")
OverflowError Overflow Error 산술 연산이 오버플로를 일으킬 때 발생합니다. raise OverflowError("error message")
RecursionError Recursion Error 재귀 한도가 초과될 때 발생합니다. raise RecursionError("error message")
ReferenceError Reference Error 약한 참조가 가리키는 객체가 이미 가비지 컬렉션 되었을 때 발생합니다. raise ReferenceError("error message")
RuntimeError Runtime Error 다른 범주에 속하지 않는 런타임 오류입니다. raise RuntimeError("error message")
StopIteration Stop Iteration 이터레이터가 더 이상 항목을 제공하지 않을 때 발생합니다. raise StopIteration("error message")
StopAsyncIteration Stop Async Iteration 비동기 이터레이터가 더 이상 항목을 제공하지 않을 때 발생합니다. raise StopAsyncIteration("error message")
SyntaxError Syntax Error 구문 오류가 발생할 때 발생합니다. raise SyntaxError("error message")
IndentationError Indentation Error 들여쓰기가 잘못될 때 발생합니다. raise IndentationError("error message")
TabError Tab Error 탭과 공백이 혼용되어 들여쓰기 오류가 발생할 때 발생합니다. raise TabError("error message")
SystemError System Error 인터프리터 내부 오류가 발생할 때 발생합니다. raise SystemError("error message")
SystemExit System Exit sys.exit() 함수가 호출될 때 발생합니다. raise SystemExit("error message")
TypeError Type Error 연산 또는 함수에 잘못된 형의 객체가 사용될 때 발생합니다. raise TypeError("error message")
UnboundLocalError Unbound Local Error 지역 변수가 참조 전에 할당되지 않았을 때 발생합니다. raise UnboundLocalError("error message")
UnicodeError Unicode Error 유니코드 관련 오류의 기본 클래스입니다. raise UnicodeError("error message")
UnicodeEncodeError Unicode Encode Error 유니코드 인코딩 오류입니다. raise UnicodeEncodeError("error message")
UnicodeDecodeError Unicode Decode Error 유니코드 디코딩 오류입니다. raise UnicodeDecodeError("error message")
UnicodeTranslateError Unicode Translate Error 유니코드 변환 오류입니다. raise UnicodeTranslateError("error message")
ValueError Value Error 연산이나 함수에 잘못된 값이 사용될 때 발생합니다. raise ValueError("error message")
ZeroDivisionError Zero Division Error 0으로 나눌 때 발생합니다. raise ZeroDivisionError("error message")

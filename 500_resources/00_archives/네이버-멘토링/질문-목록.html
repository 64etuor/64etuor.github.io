질문 목록

개발자 커리어 및 성장

  1. 개발자 여정과 성장: "멘토님은 어떻게 개발자가 되셨고, 네이버에서의 커리어 여정은 어땠나요? 주니어에서 시니어로 성장하는 과정에서 가장 중요했던 기술적/비기술적 역량은 무엇이었나요?"
  2. 경력 초반 역량: "현업에서는 신입 때 어떤 역량이나 지식을 가장 빨리 익혀야 하나요? 실무에서 바로 써먹을 수 있는 '필수 스킬'과 신입에게 가장 크게 기대하는 것은 무엇인가요?"
  3. 실패와 극복: "멘토님께서 겪었던 가장 기억에 남는 프로젝트 실패나 난관은 무엇이었고, 어떻게 극복하셨나요?"
  4. 기술 트렌드 대응: "기술 변화가 빠른 환경에서 어떻게 지속적으로 학습하고 적응하시나요? 효과적인 학습 방법이 있으신가요?"

취업 준비 및 포트폴리오

  1. 포트폴리오 전략: "취준생 입장에서 포트폴리오를 어떻게 구성해야 효율적일까요? 기업에서 특히 눈여겨보는 기술 스택이나 '이런 프로젝트는 꼭 넣어라' 같은 팁이 있다면 알려주세요."
  2. 이력서 작성 팁: "멘토님께서 많이 보시는 이력서 작성 실수나, '여기만 고치면 확 달라진다' 싶은 부분이 있을까요?"
  3. 기술 블로그: "기술 블로그를 운영할 때 기업에서 선호하는 글 스타일이나 주제가 있을까요? 효과적인 블로그 운영 방법이 궁금합니다."

네이버 특화 질문

  1. 네이버 개발 문화: "네이버의 개발 문화나 조직 분위기는 어떤가요? 다른 기업과 차별화되는 특징이 있다면 무엇인가요?"
  2. 네이버 기술 트렌드: "요즘 네이버 프론트/백엔드 팀에서 자주 쓰는 기술이 무엇인가요? 앞으로 전망도 좋다고 생각하시는 기술이 있다면 추천 부탁드려요."
  3. 기술 선택 과정: "새로운 기술이나 라이브러리를 도입할 때 네이버에서는 어떤 의사결정 과정을 거치나요? 평가 기준이나 검증 방법이 궁금합니다."

실무 협업 및 개발 프로세스

  1. 협업 과정: "프론트와 백엔드가 함께 일할 때 자주 부딪히는 지점이 있나요? 경험하신 대표적인 이슈와 대응 방법이 궁금해요."
  2. Git 협업 및 코드 리뷰: "네이버에서는 코드 리뷰와 Git 협업이 어떻게 진행되나요? '여기만큼은 꼭 본다' 하는 체크 포인트나 브랜치 전략이 있을까요?"
  3. 프로젝트 관리와 기술 부채: "네이버는 프로젝트 관리와 기술 부채를 어떻게 다루나요? 효과적이었던 프로세스나 도구가 있다면 알려주세요."

기술적 심화 질문

  1. 프론트엔드 아키텍처: "네이버의 대규모 서비스에서 프론트엔드 아키텍처는 어떻게 설계되나요? 확장성이나 유지보수를 위한 특별한 패턴이 있을까요?"
  2. 성능 최적화: "웹 성능 최적화 중에서도 네이버에서 특히 중요하게 생각하는 지표나 영역이 있나요? (초기 로딩 시간, FCP, TTI 등) 가장 효과적이었던 최적화 기법은 무엇인가요?"
  3. 테스트 전략: "프론트엔드/백엔드 테스트를 계획할 때 중점 두는 관점은 무엇인가요? 단위/통합/E2E 중 우선순위나 추천 툴이 있다면 알려주세요."

Windows 필수 프로그램 정리

📂 포맷 대비 필수 프로그램 정리

💻 노트북 드라이버

  • MyAsus
    노트북 관리 및 최적화를 위한 ASUS 전용 소프트웨어.
  • GA402RK 드라이버
    ASUS Zephyrus G14 드라이버 다운로드.
  • AMD Driver
    AMD 그래픽 및 칩셋 드라이버 다운로드 페이지.
  • G-helper
    ASUS 노트북의 팬 속도 및 전력 설정 관리.

🌐 브라우저

  • Google Chrome
    속도와 확장성이 뛰어난 브라우저.
  • Firefox
    개인 정보 보호 중심의 브라우저.

📊 프레젠테이션

  • Zoomit
    화면 확대 및 주석 작성 도구.

🎨 디자인 및 편집

  • PhotoScape X
    이미지 편집 및 디자인 소프트웨어.
  • Figma Desktop
    UI/UX 디자인 도구.
  • Framer Desktop
    웹 및 인터랙션 디자인 도구.

💻 개발 도구

  • Python
    프로그래밍 언어 Python 공식 다운로드.
  • VSCode
    오픈소스 코드 에디터.
  • Cursor AI
    AI 기반 코드 생성 도구.
  • Windsurf
    AI로 코드 편집을 지원하는 도구.
  • GitHub Desktop
    GitHub 관리용 데스크톱 앱.
  • Git Bash
    Git 명령어를 실행하는 터미널 도구.
  • Docker
    컨테이너 기반 가상화 플랫폼.

📂 데이터베이스

  • MySQL
    관계형 데이터베이스 관리 시스템.
  • PostgreSQL
    오픈소스 관계형 데이터베이스 시스템.

🤖 A.I

  • msty
    AI 기반의 텍스트 생성 및 분석 도구.

📊 대시보드

  • Power BI Desktop
    데이터 분석 및 시각화 도구.
  • Tableau
    데이터 시각화 전문 도구.

🎮 게임

  • 스팀
    게임 구매 및 관리 플랫폼.
  • 에픽 게임즈
    게임 다운로드 및 관리 플랫폼.
  • DS4Windows
    게임 컨트롤러 윈도우 지워

📚 도서

  • 밀리의 서재
    전자책 구독 서비스.

🛠️ 생산성

  • 옵시디언
    노트 및 지식 관리 도구.

🧰 유틸리티

  • Everything
    초고속 파일 검색 소프트웨어.
  • 반디집
    무료 압축/해제 도구.
  • G-hub
    Logitech 장치 관리 도구.
  • 픽픽
    스크린 캡처 및 이미지 편집 도구.

🗣️ 커뮤니티

  • 디스코드
    음성, 비디오, 텍스트 기반 커뮤니티 플랫폼.
  • 카카오톡
    메시징 및 소셜 네트워크 앱.

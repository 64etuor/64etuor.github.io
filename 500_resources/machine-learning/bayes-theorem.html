Bayes Theorem

📚 베이즈 정리 (Bayes' Theorem)

📌 정의

베이즈 정리는 확률론과 통계학에서 조건부 확률을 계산하는 데 사용하는 정리입니다. 관찰된 데이터와 사전 지식을 기반으로 사건의 가능성을 갱신하는 데 매우 유용합니다. 이는 특히 의사결정, 머신러닝, 베이지안 추론 등 다양한 응용 분야에서 중요한 역할을 합니다.

베이즈 정리는 다음과 같이 수식으로 표현됩니다:

  • : 사건 가 발생했을 때 사건 가 발생할 확률 (사후확률)
  • : 사건 가 발생했을 때 사건 가 발생할 확률 (우도)
  • : 사건 가 발생할 사전 확률 (사전확률)
  • : 사건 가 발생할 전체 확률 (정규화 상수)

🧠 핵심 개념

  1. 사전확률 (Prior Probability)

    • 사건 가 관찰 이전에 발생할 가능성을 나타냅니다.
    • 우리의 초기 신념이나 가정에 해당합니다.

  2. 우도 (Likelihood)

    • 사건 가 발생한 경우에 사건 가 발생할 가능성을 나타냅니다.

  3. 사후확률 (Posterior Probability)

    • 사건 를 관찰한 이후, 사건 에 대한 신뢰도를 갱신한 값입니다.

  4. 전체 확률 (Marginal Probability)

    • 사건 가 발생할 확률로, 모든 가능한 경우를 고려하여 계산됩니다.

🔄 베이즈 정리의 과정

  1. 사전 정보 설정

    • 사건 와 에 대한 사전확률과 우도를 설정합니다.

  2. 우도 계산

    • 관측된 데이터에 따라 값을 설정합니다.

  3. 전체 확률 계산

    • 값을 계산하여 정규화 상수를 구합니다.

  4. 사후확률 계산

    • 베이즈 정리를 통해 를 계산합니다.

🛠️ Python 코드 예제

기본 예제: 의학 진단 문제

암 진단 테스트에서 양성(true positive) 확률과 관련된 예제:

  • 암에 걸릴 확률: 암
  • 테스트가 암을 정확히 검출할 확률 (민감도): 양성암
  • 테스트가 잘못 양성(false positive)을 반환할 확률: 양성비암 
# 초기 값 설정
P_cancer = 0.01  # P(암)
P_no_cancer = 1 - P_cancer  # P(비암)

P_positive_given_cancer = 0.9  # P(양성 | 암)
P_positive_given_no_cancer = 0.05  # P(양성 | 비암)

# 전체 확률 계산
P_positive = (
    P_positive_given_cancer * P_cancer + 
    P_positive_given_no_cancer * P_no_cancer
)

# 베이즈 정리로 사후확률 계산
P_cancer_given_positive = (
    P_positive_given_cancer * P_cancer / P_positive
)

# 결과 출력
print(f"양성일 때 실제 암일 확률: {P_cancer_given_positive:.4f}")

출력 예시

양성일 때 실제 암일 확률: 0.1538

📊 시각화 예제

사전확률과 사후확률 비교

import numpy as np
import matplotlib.pyplot as plt

# 데이터 생성
x = np.linspace(0, 1, 100)
prior = 0.01
likelihood = 0.9
false_positive_rate = 0.05

# 사후확률 계산
posterior = (likelihood * prior) / (
    likelihood * prior + false_positive_rate * (1 - prior)
)

# 시각화
plt.bar(["Prior", "Posterior"], [prior, posterior], color=["blue", "orange"])
plt.title("Prior vs Posterior Probability")
plt.ylabel("Probability")
plt.ylim(0, 1)
plt.show()

🧩 응용 분야

  1. 의사결정

    • 의학 진단, 스팸 필터링, 법적 증거 해석 등 다양한 분야에서 활용.

  2. 머신러닝

    • 베이지안 네트워크, 베이지안 추론 등에서 모델의 가중치를 갱신하는 데 사용.

  3. 자연어 처리

    • 단어의 조건부 확률을 계산하여 문장 생성을 돕거나 번역 작업 수행.

  4. 비즈니스 분석

    • 고객 행동 예측, 제품 추천, A/B 테스트 결과 분석.

  5. 신호 처리

    • 노이즈가 포함된 데이터에서 유용한 정보를 추출하는 데 사용.

⚠️ 베이즈 정리 사용 시 주의사항

  1. 사전확률 설정

    • 사전확률은 결과에 큰 영향을 미칠 수 있으므로 신중히 설정해야 합니다.

  2. 우도 계산의 정확성

    • 우도가 잘못 설정되면 사후확률이 왜곡될 수 있습니다.

  3. 데이터 크기와 품질

    • 작은 샘플 크기에서는 결과가 불안정할 수 있으므로 충분한 데이터가 필요합니다.

  4. 정규화 상수 계산

    • 는 모든 가능한 경우를 고려해야 하며, 이를 간과하면 부정확한 결과를 초래할 수 있습니다.

✅ 결론

  • 베이즈 정리는 확률을 갱신하여 새로운 정보를 반영할 수 있는 강력한 도구입니다.
  • 조건부 확률 계산과 다양한 응용 분야에서 핵심적인 역할을 합니다.
  • 베이즈 정리를 통해 사전 지식과 새로운 데이터를 통합하여 더 나은 의사결정을 내릴 수 있습니다.

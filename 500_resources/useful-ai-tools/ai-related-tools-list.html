💻 Coding

Title URL Feature (특징)
gpt-engineer https://github.com/gpt-engineer-org/gpt-engineer GPT 기반 코드 생성 및 실험 플랫폼.
- 다양한 프로그래밍 언어 지원
- 자동화된 코드 골격 생성, 테스트, 리팩토링 지원
Pretzel https://github.com/pretzelai/pretzelai 대안적 오픈소스 주피터(Jupyter) 환경.
- 대규모 데이터 처리 및 ML 워크플로우 최적화
- 직관적 UI와 클라우드 기반 환경 제공
Python_Lectures https://github.com/yunho0130/Python_Lectures 파이썬 기반 강의자료 및 예제 코드 모음.
- 파이썬 기초, 데이터 처리, 알고리즘 예제 포함
- 초보 개발자 및 학생 교육용 자료
Awesome-Code-AI https://github.com/sourcegraph/awesome-code-ai AI 기반 코딩 어시스턴트, 코드 보완, 리팩토링 툴 컬렉션.
- 다양한 코딩 관련 AI 서비스를 비교, 검토 가능
- 코드 생산성 향상을 위한 참고 자료
llm https://github.com/onlybooks/llm LLM(대규모 언어 모델)을 활용한 실제 AI 애플리케이션 개발 예제.
- 해당 도서의 실습 자료 제공
- NLP 응용 서비스 개발에 참고 가능
Cursor https://www.cursor.com/ IDE형 AI
Windsurf https://codeium.com/windsurf IDE형 AI
Bolt https://bolt.new/ 풀스택 웹앱을 만들어주며, Netlify와 연동해 자동으로 Deploy까지 해주는 AI

📊 Data Analysis/BI

Title URL Feature (특징)
DeepBI https://github.com/DeepInsight-AI/DeepBI AI 주도형 데이터 분석 플랫폼.
- 대규모 데이터 셋 처리 및 인사이트 도출
- 다차원 분석 및 시각화 지원
mindsdb https://github.com/mindsdb/mindsdb AI 에이전트 및 애플리케이션을 위한 페더레이티드 쿼리 엔진.
- SQL 기반 ML 모델 예측 가능
- 다양한 데이터 소스와 손쉬운 통합
evidence https://github.com/evidence-dev/evidence SQL과 Markdown으로 대시보드, 리포트 생성이 가능한 BI 도구.
- 코드로 BI 환경 정의
- 재현성과 버전 관리 용이
Devin https://devin.ai/ AI 기반 코딩 에이전트.
- 초급 개발자 수준의 과업 수행 가능성 시험
- 향후 개발 보조 도구로 활용 가능

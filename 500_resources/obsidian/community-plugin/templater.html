📚 Templater Syntax

🔹 기본 문법

1. 변수 삽입

Templater의 기본 변수 삽입 구문은 <% %>입니다.

<% tp.file.title %>    // 파일 제목 삽입
<% tp.date.now("YYYY-MM-DD") %>    // 현재 날짜 삽입

2. 주석

Templater에서 JavaScript 주석을 사용할 수 있습니다.

<% /* 이 부분은 주석입니다 */ %>

🔹 기본 함수

1. 파일 관련 함수

함수 설명
tp.file.title 현재 파일의 제목을 반환합니다.
tp.file.path 현재 파일의 전체 경로를 반환합니다.
tp.file.creation_date 파일 생성 날짜를 반환합니다.
tp.file.last_modified_date 파일 마지막 수정 날짜를 반환합니다.

예제

파일 제목: <% tp.file.title %>  
파일 경로: <% tp.file.path %>

2. 날짜 및 시간 함수

함수 설명
tp.date.now("YYYY-MM-DD") 현재 날짜를 지정된 형식으로 반환합니다.
tp.date.now("dddd, MMMM DD, YYYY") 요일, 월, 날짜, 연도를 반환합니다.
tp.date.now("YYYY-MM-DD", -1, "days") 하루 전 날짜를 반환합니다.
tp.date.now("YYYY-MM-DD", 7, "days") 일주일 후 날짜를 반환합니다.

예제

오늘 날짜: <% tp.date.now("YYYY-MM-DD") %>  
어제 날짜: <% tp.date.now("YYYY-MM-DD", -1, "days") %>  
다음 주 날짜: <% tp.date.now("YYYY-MM-DD", 7, "days") %>

3. 사용자 입력 함수

함수 설명
tp.system.prompt("메시지") 사용자가 입력한 값을 반환합니다.
tp.system.suggester(["A", "B"], ["Option A", "Option B"]) 옵션 선택 프롬프트를 제공합니다.

예제

이름: <% tp.system.prompt("이름을 입력하세요") %>  
선택: <% tp.system.suggester(["Python", "JavaScript"], ["Python 언어", "JavaScript 언어"]) %>

🔹 고급 기능

1. 조건문

JavaScript 조건문을 사용해 동적 처리를 할 수 있습니다.

<% if (tp.date.now("dddd") === "Saturday" || tp.date.now("dddd") === "Sunday") { %>
오늘은 주말입니다!
<% } else { %>
오늘은 평일입니다.
<% } %>

2. 반복문

반복문을 사용해 리스트를 자동 생성할 수 있습니다.

<% for (let i = 1; i <= 5; i++) { %>
- 할 일 <% i %>
<% } %>

출력 결과:

- 할 일 1  
- 할 일 2  
- 할 일 3  
- 할 일 4  
- 할 일 5

3. 함수 호출

템플릿에서 사용자 정의 함수를 호출할 수 있습니다.

템플릿 파일 예제:

<%* let greet = (name) => `Hello, ${name}!`; %>
<% greet("Yangha") %>

출력 결과:

Hello, Yangha!

4. 비동기 작업

Templater는 비동기 작업도 지원합니다.

<%* let response = await fetch("https://api.github.com"); %>
<% await response.json() %>

📝 Templater 예제 템플릿

일일 노트 템플릿

---
title: "Daily Note - <% tp.date.now("YYYY-MM-DD") %>"
date: <% tp.date.now("YYYY-MM-DD") %>
tags: [daily, journal]
---

# 📅 <% tp.date.now("dddd, MMMM DD, YYYY") %>

## 📝 오늘의 할 일

<% for (let i = 1; i <= 3; i++) { %>
- [ ] <% tp.system.prompt(`할 일 ${i}을 입력하세요`) %>
<% } %>

## 🧠 오늘의 생각

<% tp.system.prompt("오늘의 주요 생각이나 깨달음을 입력하세요") %>

Type 1 vs Type 2 Decisions

두 종류의 문

Jeff Bezos가 2015년 Amazon 주주 서한에 도입한 분류는 결정을 두 종류의 문으로 비유한다.

  • Type 1 — One-way door (일방문): 되돌릴 수 없거나 되돌리는 비용이 막대한 결정. 회사 매각, 상장, 대규모 플랫폼 재설계 같은 의사결정이 여기 속한다. 한 번 통과하면 뒤로 돌아갈 수 없으므로, 충분한 심의와 합의가 필요하다.
  • Type 2 — Two-way door (양방문): 되돌릴 수 있는 결정. 실험적 기능 출시, 마케팅 캠페인 시안, 조직 구조 미세조정 등. 통과한 뒤 마음에 들지 않으면 돌아 나올 수 있다.

Bezos의 관찰은 이것이다. 조직이 커질수록 모든 결정을 Type 1처럼 다루는 경향이 생긴다. 대기업의 거버넌스는 큰 실수를 막기 위해 설계되었기 때문에, 작은 실수에도 큰 실수와 동일한 심의 비용을 부과한다. 결과는 속도의 급락이다.

Bezos는 2015년 주주 서한에서, 중대하고 비가역적인 결정(Type 1)은 신중하고 느리게, 가역적인 결정(Type 2)은 "작은 단위·개인 수준에서도 빠르게" 내리라고 권고했다. 양방문 앞에서 머뭇거리는 습관이 대기업을 더 느리게 만드는 주요 원인이라는 것이 그의 진단이다.

분류가 만드는 속도

이 분류의 힘은 대부분의 결정이 Type 2라는 사실을 재발견하게 해 주는 점에 있다. 팀이 끝없이 토론하는 안건들을 양방문/일방문으로 분류하면, 상당수가 "실행하고 2주 뒤 평가"로 넘어갈 수 있다. 토론의 깊이와 길이를 결정의 종류에 맞춰 조정하는 거버넌스 도구가 된다.

실수의 비대칭

Bezos는 Type 1 결정을 Type 2로 취급하는 실수(무모함)와 Type 2 결정을 Type 1으로 취급하는 실수(느림) 사이에 비대칭적 비용이 있다고 봤다. 작은 조직에서는 후자의 비용이 보통 더 크고, 큰 조직에서는 전자의 비용이 더 커진다. 조직의 단계에 따라 기본값을 다르게 설정해야 한다는 함의다.

한계와 보완

Type 1/Type 2는 단순해서 강력하지만, 실무에서는 가역성(reversibility), 규모(scale), 시급성(urgency) 의 3축으로 확장해 보는 경우가 많다. 되돌릴 수 있어도 규모가 크면 신중해야 하고, 되돌릴 수 없어도 시급하면 빠르게 결정해야 한다.

또한 "되돌릴 수 있다"는 판정 자체가 종종 과소평가된다. 브랜드 신뢰, 조직 문화, 핵심 인재의 마음처럼 표면적으로 되돌릴 수 있어 보이지만 실질적으로는 Type 1에 가까운 결정이 있다. 판정의 정직함이 이 프레임의 유효성을 결정한다.

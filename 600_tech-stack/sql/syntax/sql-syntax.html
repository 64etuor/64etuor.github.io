SQL Syntax

🧠 Understanding SQL Syntax

📌 Introduction

SQL (Structured Query Language) is a standard language for interacting with relational databases. SQL is used to perform operations such as querying data, updating records, and managing database objects. Understanding SQL syntax is crucial for writing clear, efficient, and error-free SQL statements.

📜 SQL Statement Structure

SQL statements generally follow this structure:

SELECT column1, column2, ...
FROM table_name
WHERE condition
GROUP BY column
HAVING condition
ORDER BY column ASC | DESC;

📝 Common Clauses in SQL

  1. SELECT: Specifies the columns to retrieve.
  2. FROM: Specifies the table(s) to retrieve data from.
  3. WHERE: Filters rows based on specified conditions.
  4. GROUP BY: Groups rows that have the same values into summary rows.
  5. HAVING: Filters grouped rows based on conditions.
  6. ORDER BY: Sorts the result set.

🗂️ SQL Keywords

SQL keywords are case-insensitive, but it's a common practice to write them in uppercase for readability.

Keyword Description
SELECT Retrieves data from a database.
FROM Specifies the table to query data from.
WHERE Filters rows based on a condition.
INSERT INTO Adds new rows to a table.
UPDATE Modifies existing rows in a table.
DELETE Removes rows from a table.
JOIN Combines rows from two or more tables.
GROUP BY Groups rows that share the same values.
HAVING Filters groups based on conditions.
ORDER BY Sorts the results in ascending or descending order.
CREATE Creates a new database object (e.g., table, index).
ALTER Modifies existing database objects.
DROP Deletes database objects.
TRUNCATE Removes all rows from a table (faster than DELETE).
UNION Combines the results of two or more SELECT statements.

🔍 Basic SQL Statements

1. SELECT Statement

Retrieves data from a table.

SELECT FirstName, LastName
FROM Employees
WHERE Department = 'Sales';

2. INSERT Statement

Adds new rows to a table.

INSERT INTO Employees (FirstName, LastName, Department, Salary)
VALUES ('Alice', 'Smith', 'HR', 60000);

3. UPDATE Statement

Modifies existing rows in a table.

UPDATE Employees
SET Salary = 70000
WHERE EmployeeID = 1;

4. DELETE Statement

Removes rows from a table.

DELETE FROM Employees
WHERE EmployeeID = 1;

📝 Clauses and Their Usage

1. WHERE Clause

Filters rows based on a condition.

SELECT * FROM Products
WHERE Price > 100;

2. GROUP BY Clause

Groups rows with the same values.

SELECT Department, COUNT(*) AS EmployeeCount
FROM Employees
GROUP BY Department;

3. HAVING Clause

Filters groups based on conditions (used with GROUP BY).

SELECT Department, COUNT(*) AS EmployeeCount
FROM Employees
GROUP BY Department
HAVING COUNT(*) > 5;

4. ORDER BY Clause

Sorts results in ascending or descending order.

SELECT FirstName, LastName, Salary
FROM Employees
ORDER BY Salary DESC;

🔄 Combining Queries

1. UNION Operator

Combines results of two or more SELECT statements (removes duplicates).

SELECT FirstName FROM Employees
UNION
SELECT FirstName FROM Customers;

2. UNION ALL Operator

Combines results without removing duplicates.

SELECT FirstName FROM Employees
UNION ALL
SELECT FirstName FROM Customers;

🔧 SQL Data Types

Data Type Description Example
INT Integer values 42
VARCHAR(n) Variable-length string (up to n chars) 'Alice'
CHAR(n) Fixed-length string 'HR'
FLOAT Floating-point numbers 3.14
DATE Date value '2024-01-01'
BOOLEAN True or False TRUE, FALSE

🔒 SQL Constraints

Constraint Description
PRIMARY KEY Uniquely identifies each row in a table.
FOREIGN KEY Ensures referential integrity between tables.
NOT NULL Ensures a column cannot have NULL values.
UNIQUE Ensures all values in a column are unique.
CHECK Ensures values meet a specific condition.
DEFAULT Sets a default value if none is provided.

Example

CREATE TABLE Employees (
    EmployeeID INT PRIMARY KEY,
    FirstName VARCHAR(50) NOT NULL,
    Salary FLOAT CHECK (Salary > 0)
);

📝 SQL Operators

Comparison Operators

Operator Description Example
= Equal Salary = 50000
<>, != Not equal Salary != 50000
> Greater than Salary > 50000
< Less than Salary < 50000
>= Greater or equal Salary >= 50000
<= Less or equal Salary <= 50000

Logical Operators

Operator Description Example
AND Both conditions are true Salary > 50000 AND Department = 'HR'
OR At least one condition is true Department = 'HR' OR Department = 'Sales'
NOT Negates a condition NOT (Salary > 50000)

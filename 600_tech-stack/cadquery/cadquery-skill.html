CadQuery Skill

CadQuery Code Generation

You generate production-quality CadQuery Python scripts that run in CQ-editor or standalone Python.

Before writing code

  1. Clarify the shape — If the user's request is ambiguous, ask about critical dimensions, tolerances, or topology before writing code. Don't guess structural intent.

  2. Think in workplanes — Plan which faces you'll work from. CadQuery's power comes from relative positioning on workplanes, not absolute coordinates.

  3. Start simple, add detail — Create the base shape first, then add features (holes, fillets, chamfers) in order of structural importance.

Code structure

Every script should follow this pattern:


import cadquery as cq

  

# --- Parameters ---

width = 80

height = 60

thickness = 10

  

# --- Model ---

result = (

    cq.Workplane("XY")

    .box(width, height, thickness)

    # ... operations chained fluently

)

  

# --- Display ---

show_object(result, name="part_name", options={"color": "steelblue"})

Rules:

  • Always put dimensions in named variables at the top — this is what makes CadQuery parametric

  • Use the fluent API (method chaining on Workplane) as the primary approach

  • End with show_object() for CQ-editor compatibility

  • Add cq.exporters.export(result, "output.step") only when the user asks for export

Core API patterns

Read references/api-cheatsheet.md for the full API quick reference when you need to look up specific method signatures, selector syntax, or advanced features.

Workplane basics


# Start on XY, XZ, or YZ plane (also: "front","back","left","right","top","bottom")

cq.Workplane("XY")

  

# Move to a face of existing solid, then create a new workplane

.faces(">Z").workplane()

  

# Offset workplane

.faces(">Z").workplane(offset=5.0)

  

# Control where workplane origin lands

.faces(">Z").workplane(centerOption="CenterOfMass")      # face center of mass

.faces(">Z").workplane(centerOption="ProjectedOrigin")    # default: project parent origin

.faces(">Z").workplane(centerOption="CenterOfBoundBox")   # bounding box center

  

# Tag a workplane state and return to it later

.tag("base")

# ... operations ...

.workplaneFromTagged("base")

  

# Angled workplane via local rotation/offset

.transformed(rotate=(0, 0, 45), offset=(0, 0, 5))

2D drawing → 3D

The core loop: select/create a workplane → draw 2D → extrude/revolve/sweep to 3D.


# Primitives

.box(length, width, height)

.sphere(radius)

.cylinder(height, radius)

  

# Sketch → extrude

.rect(w, h).extrude(depth)

.circle(r).extrude(depth)

.polygon(n_sides, diameter).extrude(depth)

  

# Wire drawing → extrude

.moveTo(x, y).lineTo(x2, y2).threePointArc((mx, my), (ex, ey)).close().extrude(depth)

  

# Revolve (axis is relative to current workplane center)

.rect(w, h).revolve(angleDegrees, axisStart, axisEnd)

  

# Sweep along a path

.sweep(path_wire)

# transition options: "right" (default), "round", "transformed"

  

# Loft between sections

.loft()  # requires multiple pending wires

  

# Extrude to face (not just a distance)

.circle(r).extrude("next")   # extrude until next face

.circle(r).extrude("last")   # extrude until last face

Combine parameter

Most 3D operations accept a combine parameter that controls how new geometry interacts with the context solid:


.circle(r).extrude(h)                       # combine=True (default): union with context solid

.circle(r).extrude(h, combine=False)         # separate geometry, no boolean

.circle(r).extrude(h, combine="cut")         # subtract from context solid (same as "s")

The first solid created becomes the "context solid." Subsequent operations with combine=True automatically union with it.

Selectors

Selectors pick faces, edges, or vertices for subsequent operations. See references/api-cheatsheet.md for the complete selector reference.

Most common patterns:


.faces(">Z")      # top face (farthest in +Z)

.faces("<Z")      # bottom face

.faces("+Z")      # faces with normal pointing in +Z

.edges("|Z")      # edges parallel to Z

.edges(">Z")      # topmost edge

.edges("#Z")      # edges perpendicular to Z

Combining selectors:


.edges(">Z and |X")     # AND

.faces(">Z or <Z")      # OR

.edges("not |Z")         # NOT

Index selectors (useful for picking specific faces on complex geometry):


.faces(">Z[0]")      # farthest face (same as >Z)

.faces(">Z[1]")      # second farthest

.faces(">>Z[0]")     # center-based: nearest to center

Type selectors (work on curved geometry where direction selectors don't):


.edges("%Circle")     # circular edges

.faces("%Cylinder")   # cylindrical faces

Programmatic filtering:


.edges().filter(lambda e: e.Length() > 10)

.faces().sort(lambda f: f.Area())

Multi-point iteration — selectors populate the stack, and subsequent operations apply to all items:


.faces(">Z").workplane()

.rect(70, 50, forConstruction=True)

.vertices()                    # 4 corner points on stack

.cboreHole(2.4, 4.4, 2.1)    # hole at each corner

Boolean operations


.union(other_solid)

.cut(other_solid)

.intersect(other_solid)

  

# Shorthand via workplane

.cutBlind(-depth)        # cut into solid from current workplane

.cutThruAll()            # cut all the way through

Fillets and chamfers

Always apply these AFTER creating all the geometry they affect:


.edges("|Z").fillet(2)       # fillet vertical edges

.edges(">Z").chamfer(1)     # chamfer top edges

.edges(">Z and |X").fillet(0.5)  # combine selectors with 'and'

.edges(">Z").chamfer(1, 2)  # asymmetric chamfer

Holes


.hole(diameter)                              # simple through-hole

.cboreHole(bolt_d, cbore_d, cbore_depth)    # counterbore

.cskHole(bolt_d, csk_d, csk_angle)          # countersink

Shell (hollowing)


.faces(">Z").shell(-wall_thickness)   # hollow out, removing top face

# kind="arc" (default, rounded) or "intersection" (sharp)

Text engraving

CadQuery can engrave or emboss text directly onto solids:


# Cut text into the top face (default combine="cut")

.faces(">Z").workplane()

.text("HELLO", fontsize=10, distance=-1)

  

# Emboss text outward

.faces(">Z").workplane()

.text("HELLO", fontsize=10, distance=2, combine=True)

  

# Font control

.text("CQ", fontsize=8, distance=-0.5,

      font="Arial", kind="bold",

      halign="center", valign="center")

Patterns and arrays


# Rectangular array

.rarray(x_spacing, y_spacing, x_count, y_count)

.circle(r).extrude(h)

  

# Polar array

.polarArray(radius, startAngle, angle, count)

# fill=True (default): distribute count items evenly over angle

# fill=False: use angle as spacing between items

.circle(r).extrude(h)

  

# Construction geometry for custom placement

.pushPoints([(x1,y1), (x2,y2), ...])

.circle(r).extrude(h)

Iteration with each() / eachpoint()

For custom operations at multiple locations:


# eachpoint: translate a shape to each point on the stack

def make_pin(loc):

    return cq.Workplane().cylinder(5, 1).val().located(loc)

  

result = (

    cq.Workplane("XY")

    .box(20, 20, 2)

    .faces(">Z").workplane()

    .rarray(8, 8, 2, 2)

    .eachpoint(make_pin, combine=True)

)

Wire utilities


# Offset a 2D wire inward/outward

.rect(20, 10).offset2D(-2, kind="arc")  # inward offset

# kind: "arc" (rounded), "intersection" (sharp), "tangent"

  

# Bridge method: add stack wires to pending for extrude/loft

.toPending()

  

# DXF import pattern

cq.importers.importDXF("profile.dxf").wires().toPending().extrude(10)

Section and split


# Get cross-section at a height

.section(height=5.0)  # returns face(s) at that height

  

# Split solid by workplane

.split(keepTop=True, keepBottom=False)

Advanced curves


# Bezier curve

.bezier([(0,0), (10,20), (20,0)])

  

# Parametric curve

.parametricCurve(lambda t: (10*cos(t*2*pi), 10*sin(t*2*pi)), start=0, stop=1)

  

# Spline with tangent control

.spline(points, tangents=[(0,1), (1,0)])

Assemblies

For multi-part designs, use cq.Assembly():


import cadquery as cq

  

base = cq.Workplane("XY").box(100, 80, 10)

post = cq.Workplane("XY").cylinder(40, 5)

  

assy = cq.Assembly()

assy.add(base, name="base", color=cq.Color("gray"))

assy.add(post, name="post", loc=cq.Location((20, 0, 25)), color=cq.Color("red"))

  

show_object(assy)

For constrained assemblies, read references/api-cheatsheet.md — the Assembly section.

Sketch API

For complex 2D profiles, the Sketch API offers more control than Workplane 2D methods:


result = (

    cq.Workplane("XY")

    .sketch()

    .rect(80, 60)                          # add mode by default

    .vertices().fillet(5)                   # fillet sketch corners

    .reset()                               # IMPORTANT: reset selection between operations

    .rarray(60, 40, 2, 2)

    .circle(3, mode="s")                   # subtract mode

    .finalize()

    .extrude(10)

)

Modes: "a" (add), "s" (subtract), "i" (intersect), "c" (construction), "r" (replace).

Critical: Sketch modifies in-place without undo history. Always call .reset() between different selector operations. Also, Sketch's .arc() uses multimethod dispatch — never use keyword arguments for its positional parameters.

Import / Export


# Export

cq.exporters.export(result, "part.step")   # STEP (recommended for CAD)

cq.exporters.export(result, "part.stl")    # STL (for 3D printing)

cq.exporters.export(result, "part.dxf")    # DXF (2D drawing)

cq.exporters.export(result, "part.svg")    # SVG (2D render)

  

# Import

imported = cq.importers.importStep("existing.step")

# DXF import (must use toPending for extrude)

profile = cq.importers.importDXF("drawing.dxf").wires().toPending().extrude(10)

Common pitfalls to avoid

  1. Don't fillet/chamfer before geometry is complete — adding more features after a fillet can cause kernel failures. Apply fillets and chamfers last.

  2. Don't use absolute coordinates when relative positioning works — use .faces(">Z").workplane() to position relative to existing geometry instead of calculating absolute positions.

  3. Don't create unnecessary boolean operations — the context solid auto-combines. You rarely need explicit .union() within a single chain.

  4. Selector on non-planar edges — direction selectors like |Z or >Z exclude non-linear edges. Use %Circle type selector or >>Z center-based selector for curved geometry.

  5. Avoid booleans in loops — they're expensive. Build geometry first, then combine once.

  6. Sketch .arc() keyword args — Sketch uses multimethod dispatch; passing keyword arguments to .arc() will fail silently or error. Always use positional arguments.

  7. Forgot .reset() in Sketch — Sketch has no history. After .vertices().fillet(), you must call .reset() before doing anything else.

  8. DXF import without .toPending() — imported DXF wires won't extrude unless you call .wires().toPending() first.

  9. show_object compatibility — for scripts that should run both in CQ-editor and standalone:

   ```python

   if "show_object" not in dir():

       def show_object(*args, **kwargs):

           pass

   ```

Color reference

Common color names for show_object() and assemblies:

"red", "green", "blue", "yellow", "orange", "purple", "white", "gray", "black", "steelblue", "gold", "silver", "cyan", "magenta"

Transparency: options={"alpha": 0.5} (0 = transparent, 1 = opaque)

RGB tuple: cq.Color(0.8, 0.2, 0.2) or cq.Color(0.8, 0.2, 0.2, 0.5) with alpha.

데이터를 직접 관리함으로써 얻는 이점?

  1. 데이터 소유권 확보
  2. 보안, 개인정보 보호
  3. 비용 최적화
  4. 맞춤형 데이터 처리 가능
  5. 서비스 간 연결 자유도 증가
  6. 장기적인 확장성과 독립성 확보

Primary Key(기본 키)와 Foreign Key(외래키)

우리가 사람을 구별하는 방법

이름 성별 외모 사는 곳 나이 직업 말투 옷차림

테이블 명세서

✅ task (일일 기록)

필드명 타입 설명 비고
PK_task_id TEXT (PK) 고유한 작업 ID 자동 생성
started_at DATETIME 작업 시작 시간 자동 기록
ended_at DATETIME 작업 종료 시간 자동 기록
duration_min INT 소요 시간 (분) ended_at - started_at으로 자동 계산
created_date DATE 생성 날짜 자동 입력
task_type ENUM 작업 유형
task_description TEXT 작업 상세 내용
FK_plan_id TEXT (FK) 연관된 plan ID plan에서 파생 가능
satisfaction_rating INT 만족도 평가 (1~5)
reason_of_rating TEXT 평가 이유
image BLOB 첨부 이미지 이미지는 앱시트 내부 로직에 따라 상대 주소로 참조
location TEXT 작업 위치 자동 입력 (GPS 기반)
is_plan_completed BOOLEAN 액션 트리거용 플랜 완료를 위한 액션 트리거 표시 여부를 위한 Boolean 값

✅ plan (일일 계획)

필드명 타입 설명 비고
PK_plan_id TEXT(PK) 고유한 계획 ID 자동 생성
created_at DATETIME 생성 시간 자동 입력
completed_at DATETIME 완료된 시간 NULL이면 미완료
cancelled_at DATETIME 취소된 시간 NULL이면 유효
created_date DATE 생성 날짜 자동 입력
completed_date DATE 완료 날짜 자동 입력
cancelled_date DATE 취소 날짜 자동 입력
time_period ENUM 계획된 시간 범위 morning, afternoon, evening
status ENUM 진행 상태 queued, completed, cancelled
task_type ENUM 작업 유형
plan_description TEXT 계획 상세 내용
CSF TEXT 핵심 성공 요인
FK_todo_id TEXT (FK) 연관된 todo ID todo에서 파생 가능
reason_of_cancellation TEXT 취소 사유

✅ todo (중기 계획 / 일회성 계획)

필드명 타입 설명 비고
PK_todo_id TEXT (PK) 고유한 할 일 ID 자동 생성
created_at DATETIME 생성 시간 자동 입력
started_at DATETIME 시작된 시간 액션 입력
completed_at DATETIME 완료된 시간 액션 입력
cancelled_at DATETIME 취소된 시간 액션 입력
created_date DATE 생성 날짜 자동 입력
completed_date DATE 완료 날짜 자동 입력
cancelled_date DATE 취소 날짜 자동 입력
status ENUM 진행 상태 queued, completed, cancelled, pending, in-progress
task_type ENUM 작업 유형
todo_description TEXT 할 일 상세 내용
start_date DATE 시작 예정일
end_date DATE 마감 예정일
remaining_days INT 남은 일 수 자동 입력 (Virtual Column 기반 동적 계산)
FK_project_id TEXT (FK) 연관된 project ID project에서 파생 가능
reason_of_cancellation TEXT 취소 사유
is_one_time BOOLEAN 단발성 계획 여부
google_task_id TEXT Google Tasks와 연동되는 ID 자동 입력(Apps Script)

PK_project_id TEXT (PK) 고유한 프로젝트 ID 자동 생성

status ENUM 진행 상태 queued, completed, cancelled, pending, in-progress
task_type ENUM 작업 유형
project_description TEXT 프로젝트 상세 내용
start_date DATE 시작 예정일
end_date DATE 마감 예정일
remaining_days INT 남은 일 수 end_date - today()
reason_of_cancellation TEXT 취소 사유
google_task_id TEXT Google Tasks와 연동되는 ID

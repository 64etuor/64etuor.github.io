Linux 표준입출력

Linux 기호(특수 문자) 사용법 정리

1. 일반적인 기호

기호 설명 예제
# 주석(Comment) # 이 줄은 주석입니다
$ 일반 사용자 프롬프트 $ ls -l
~ 현재 사용자의 홈 디렉터리 cd ~
. 현재 디렉터리 ls .
.. 상위 디렉터리 cd ..
- 이전 디렉터리 cd -

2. 와일드카드(Wildcards)

기호 설명 예제
* 모든 문자와 일치 (0개 이상) ls *.txt → .txt 확장자를 가진 모든 파일 표시
? 단일 문자와 일치 ls file?.txt → file1.txt, file2.txt 등과 일치
[ ] 문자 집합과 일치 ls file[12].txt → file1.txt, file2.txt 일치
{ } 여러 패턴과 일치 (쉘 확장) echo {a,b,c} → a b c

3. 리디렉션(Redirection)

기호 설명 예제
> 출력 리디렉션 (덮어쓰기) echo "Hello" > file.txt
>> 출력 리디렉션 (추가) echo "World" >> file.txt
< 입력 리디렉션 wc -l < file.txt
2> 오류 메시지 리디렉션 command 2> error.log
&> 표준 출력과 오류 출력 함께 리디렉션 command &> output.log
` ` 파이프(Pipe)

4. 백그라운드 실행 및 작업 제어

기호 설명 예제
& 백그라운드 실행 command &
jobs 백그라운드 작업 목록 표시 jobs
fg 백그라운드 작업을 포그라운드로 이동 fg %1
bg 백그라운드에서 다시 실행 bg %1
Ctrl + Z 실행 중인 작업을 중지 실행 중인 프로그램에서 Ctrl + Z 입력

5. 논리 연산자 및 조건문

기호 설명 예제
&& 명령어가 성공한 경우 다음 명령 실행 mkdir test && cd test
` `
; 순차적으로 명령어 실행 cd /tmp; ls -l

6. 변수 및 문자열 관련

기호 설명 예제
$var 변수 참조 echo $HOME
${var} 변수 확장 echo ${USER}
$(command) 명령어 실행 결과 치환 echo $(date)
`command` 백틱(Backtick)으로 명령어 실행 echo `date`
" 변수 및 명령어 치환 허용 echo "$USER"
' 리터럴 문자열(치환 없음) echo '$USER'

7. 특수 문자 이스케이프

기호 설명 예제
\ 특수 문자 이스케이프 echo \"Hello\"
\n 줄바꿈(Newline) echo -e "Hello\nWorld"
\t 탭(Tab) echo -e "Hello\tWorld"

8. 프로세스 및 파일 설명자

기호 설명 예제
$! 마지막 실행한 백그라운드 프로세스 ID sleep 10 & echo $!
$$ 현재 쉘의 프로세스 ID echo $$
$? 이전 명령의 종료 상태 코드 ls file.txt; echo $?

9. 파일 권한 및 실행 관련

기호 설명 예제
+ 권한 추가 chmod +x script.sh
- 권한 제거 chmod -r file.txt
= 정확한 권한 설정 chmod 644 file.txt
` ` 파이프를 통한 명령어 연결

10. 프로세스 및 실행 관련

기호 설명 예제
() 서브쉘 실행 (cd /tmp; ls)
{} 여러 명령어 그룹 실행 { date; whoami; }
exec 현재 쉘을 다른 프로그램으로 대체 exec bash

예제 활용

# home 디렉터리에 있는 .txt 파일들을 모두 새로운 폴더로 이동
mkdir backup && mv ~/*.txt backup/

# test 디렉터리가 없으면 만들고 이동
[ -d test ] || mkdir test && cd test

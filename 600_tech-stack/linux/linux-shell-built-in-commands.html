Linux Shell built-in commands

알겠어! 쉘 명령어(Shell Built-in Commands)와 쉘 스크립트 관련 명령어를 따로 정리할게.

🚀 리눅스 쉘 명령어 정리 (Shell Built-in Commands)

쉘 명령어는 일반적인 리눅스 명령어(ls, cp 등)와는 다르게 쉘 내부에서 실행되는 명령어야.
즉, 별도의 실행 파일이 아니라 쉘 자체에 내장된 기능을 수행하는 명령어들이야.

📌 쉘 명령어 확인 방법

type 명령어

예제:

type cd   # cd is a shell builtin
type ls   # ls is /bin/ls (외부 명령어)

cd는 쉘 내부 명령어이고, ls는 실행 파일임을 알 수 있어.

🔹 1. 셸 환경 관리 명령어

명령어 설명
echo $SHELL 현재 사용 중인 쉘 확인 (예: /bin/bash)
set 현재 설정된 모든 환경 변수 출력
env 현재 환경 변수 목록 출력
export VAR=value 환경 변수 설정 (export PATH=$PATH:/new/path)
unset VAR 환경 변수 삭제
alias ll='ls -l' 명령어 단축어(alias) 설정
unalias ll alias 제거
history 명령어 기록 확인
!! 마지막 명령어 다시 실행
!n 히스토리에서 n번째 명령어 실행 (!10)

📌 💡 예제

export MYVAR="Hello, Shell!"  # 환경 변수 설정
echo $MYVAR  # 변수 출력
unset MYVAR  # 변수 삭제

🔹 2. 디렉토리 및 파일 관련 쉘 명령어

명령어 설명
cd 디렉토리 이동 (cd /home/user)
cd - 이전 디렉토리로 이동
pwd 현재 디렉토리 출력
pushd [디렉토리] 디렉토리 스택에 현재 위치 저장 후 이동
popd 스택에서 디렉토리 가져와 이동
dirs 디렉토리 스택 확인

📌 💡 예제

pushd /var/log  # 현재 디렉토리를 스택에 저장하고 /var/log로 이동
popd  # 이전 디렉토리로 복귀

🔹 3. 프로세스 관리 관련 쉘 명령어

명령어 설명
jobs 백그라운드 작업 목록 확인
fg 백그라운드 작업을 포그라운드로 가져옴
bg 중지된 작업을 백그라운드에서 계속 실행
kill %n 특정 백그라운드 작업 종료 (kill %1)
wait 백그라운드 작업이 끝날 때까지 대기
disown 백그라운드 작업을 현재 쉘에서 떼어냄 (종료해도 유지됨)

📌 💡 예제

sleep 100 &  # 백그라운드에서 실행
jobs  # 실행 중인 작업 확인
fg %1  # 첫 번째 백그라운드 작업을 포그라운드로 가져오기
kill %1  # 해당 작업 종료

🔹 4. 조건문 및 반복문 (쉘 스크립트에서 사용)

명령어 설명
if [ 조건 ]; then ... fi 조건문 실행
[ -f 파일명 ] 파일 존재 여부 확인
[ -d 디렉토리 ] 디렉토리 존재 여부 확인
for 변수 in 리스트; do ... done for 루프 실행
while [ 조건 ]; do ... done while 루프 실행
case $변수 in 값1) ... ;; esac case 조건문 실행

📌 💡 예제

if [ -f myfile.txt ]; then
  echo "파일이 존재합니다"
else
  echo "파일이 없습니다"
fi

🔹 5. 입력/출력 리다이렉션 관련 쉘 명령어

명령어 설명
> 표준 출력을 파일로 저장 (덮어쓰기)
>> 표준 출력을 파일로 저장 (이어쓰기)
2> 표준 에러를 파일로 저장
&> 표준 출력과 표준 에러를 모두 저장
< 표준 입력을 파일에서 읽음
` `

📌 💡 예제

echo "Hello" > output.txt  # 파일에 저장
ls /notexist 2> error.log  # 에러 메시지를 error.log에 저장
cat file.txt | grep "hello"  # 파일에서 "hello" 포함된 줄 찾기

🔹 6. 쉘 스크립트 작성 예제

간단한 Bash 스크립트 작성법도 포함할게!

✅ 1. Hello World 출력

#!/bin/bash
echo "Hello, World!"

저장 후 실행 권한 부여:

chmod +x script.sh
./script.sh

✅ 2. 사용자 입력 받기

#!/bin/bash
echo "이름을 입력하세요:"
read name
echo "반갑습니다, $name!"

✅ 3. 반복문을 활용한 자동 파일 생성

#!/bin/bash
for i in {1..5}; do
  touch "file_$i.txt"
done

→ file_1.txt ~ file_5.txt 자동 생성됨.

✅ 4. 특정 폴더의 파일 개수 세기

#!/bin/bash
dir="/home/user"
count=$(ls "$dir" | wc -l)
echo "$dir 안에 $count 개의 파일이 있습니다."

✅ 5. 특정 프로세스 종료 자동화

#!/bin/bash
echo "종료할 프로세스 이름을 입력하세요:"
read pname
killall "$pname"
echo "$pname 프로세스를 종료했습니다."

🎯 결론: 꼭 알아야 할 쉘 명령어 정리

🔥 리눅스를 다룰 때 필수적인 쉘 명령어 

echo "Hello" > file.txt  # 파일 쓰기
cat file.txt | grep "word"  # 특정 단어 검색
cd /var/log && ls -l  # 로그 디렉토리 이동 후 파일 목록 확인
pushd /etc && popd  # 이전 디렉토리 스택 이동
export PATH=$PATH:/custom/path  # 환경 변수 추가
alias ll='ls -l'  # 명령어 단축어 설정
jobs  # 백그라운드 작업 확인
fg %1  # 백그라운드 작업을 다시 활성화
for i in {1..5}; do echo "파일 $i"; done  # 루프 실행

이제 쉘 명령어와 기본 스크립팅까지 마스터하면 리눅스를 훨씬 더 유용하게 활용할 수 있어! 🚀💻

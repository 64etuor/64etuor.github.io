Linux Commands

🚀 리눅스에서 자주 사용하는 명령어 정리

리눅스를 사용할 때 가장 많이 쓰이는 기본 명령어부터 고급 명령어까지 정리해봤어! 💻

🔹 1. 기본적인 파일 및 디렉터리 관리

명령어 설명
pwd 현재 경로 출력 (Print Working Directory)
ls 현재 디렉터리의 파일 목록 표시
ls -l 자세한 파일 정보 출력 (권한, 크기, 날짜 등)
ls -a 숨김 파일까지 모두 표시
cd [경로] 디렉터리 이동 (예: cd /home)
cd .. 한 단계 위 디렉터리로 이동
cd - 이전 디렉터리로 이동
mkdir [폴더명] 새 디렉터리 생성
rmdir [폴더명] 빈 디렉터리 삭제
rm -r [폴더명] 디렉터리 및 내부 파일 삭제
rm -f [파일명] 파일 강제 삭제 (확인 없이)
cp [파일] [복사위치] 파일 복사
mv [파일] [새이름 또는 이동위치] 파일 이동 또는 이름 변경

📌 💡 예제: 

mkdir my_project  # my_project 폴더 생성
cd my_project     # 해당 폴더로 이동
touch test.txt    # 빈 파일 생성
mv test.txt renamed.txt  # 파일 이름 변경
rm renamed.txt  # 파일 삭제

🔹 2. 파일 및 텍스트 처리

명령어 설명
cat [파일명] 파일 내용 출력
less [파일명] 긴 파일을 페이지 단위로 출력 (↑↓ 스크롤 가능)
head -n 10 [파일명] 파일의 처음 10줄 출력
tail -n 10 [파일명] 파일의 마지막 10줄 출력
tail -f [로그파일] 실시간 로그 모니터링
nano [파일명] nano 편집기로 파일 수정
vim [파일명] vim 편집기로 파일 수정
grep "문자열" [파일명] 특정 문자열 검색
grep -r "문자열" [폴더] 폴더 내 모든 파일에서 문자열 검색

📌 💡 예제: 

cat /etc/passwd  # 사용자 계정 정보 출력
grep "root" /etc/passwd  # 파일에서 'root' 포함된 줄 찾기
tail -f /var/log/syslog  # 시스템 로그 실시간 모니터링

🔹 3. 프로세스 및 시스템 관리

명령어 설명
ps aux 실행 중인 프로세스 목록 보기
top 실시간 시스템 상태 확인 (CPU, RAM 사용량 등)
htop top보다 개선된 인터페이스 (설치 필요)
kill [PID] 특정 프로세스 종료
kill -9 [PID] 강제 종료 (SIGKILL)
pkill [이름] 특정 프로세스를 이름으로 종료
free -h 메모리 사용량 확인
uptime 시스템 부팅 후 경과 시간 확인
df -h 디스크 사용량 확인
du -sh [폴더] 특정 폴더의 크기 확인

📌 💡 예제: 

ps aux | grep firefox  # 실행 중인 Firefox 프로세스 찾기
kill -9 12345  # PID 12345 강제 종료
df -h  # 디스크 사용량 확인
free -h  # 메모리 사용량 확인

🔹 4. 사용자 및 권한 관리

명령어 설명
whoami 현재 로그인한 사용자 확인
id 현재 사용자의 UID, GID 확인
who 현재 로그인한 모든 사용자 표시
su [사용자] 다른 사용자 계정으로 전환
sudo [명령어] 관리자(root) 권한으로 명령 실행
chmod 755 [파일] 파일 권한 변경 (소유자는 읽기/쓰기/실행, 그룹 및 기타 사용자는 읽기/실행 가능)
chown user:group [파일] 파일 소유자 변경
passwd [사용자] 비밀번호 변경

📌 💡 예제: 

sudo apt update  # 패키지 목록 업데이트 (root 권한 필요)
chmod 644 myfile.txt  # 파일을 읽기 전용으로 설정
chown ubuntu:ubuntu myfile.txt  # 파일 소유자 변경

🔹 5. 네트워크 및 인터넷 관련 명령어

명령어 설명
ping google.com 네트워크 연결 확인
curl -I google.com HTTP 헤더 확인
wget [URL] 파일 다운로드
netstat -tulnp 현재 열려 있는 포트 확인
ss -tulnp netstat 대체 명령어 (더 빠름)
ifconfig 또는 ip a 네트워크 인터페이스 정보 확인
nslookup google.com DNS 조회
traceroute google.com 네트워크 경로 추적

📌 💡 예제: 

ping 8.8.8.8  # 구글 DNS 서버와 연결 확인
wget https://example.com/file.zip  # 파일 다운로드
curl -I https://www.naver.com  # 웹사이트 응답 확인

🔹 6. 압축 및 아카이브 관련 명령어

명령어 설명
tar -cvf archive.tar [파일/폴더] tar 압축 파일 생성
tar -xvf archive.tar tar 파일 해제
tar -czvf archive.tar.gz [파일/폴더] gzip 압축 파일 생성
tar -xzvf archive.tar.gz gzip 압축 해제
zip -r archive.zip [파일/폴더] zip 압축
unzip archive.zip zip 압축 해제

📌 💡 예제: 

tar -czvf mybackup.tar.gz /home/user  # 홈 디렉토리 압축
unzip mydocs.zip  # zip 파일 해제

🎯 리눅스에서 가장 많이 쓰는 명령어 TOP 10

🔥 리눅스를 다룰 때 가장 기본적인 명령어들! 

pwd             # 현재 위치 확인
ls -l          # 디렉터리 목록 자세히 보기
cd /path       # 디렉터리 이동
rm -rf folder  # 폴더 삭제 (주의!)
cat file.txt   # 파일 내용 보기
grep "word" file.txt  # 파일에서 특정 단어 검색
ps aux | grep process  # 실행 중인 프로세스 찾기
df -h          # 디스크 사용량 확인
sudo apt update && sudo apt upgrade -y  # 시스템 패키지 업데이트
ping google.com  # 네트워크 상태 확인

이제 리눅스를 더 효율적으로 활용할 수 있을 거야! 🚀💻

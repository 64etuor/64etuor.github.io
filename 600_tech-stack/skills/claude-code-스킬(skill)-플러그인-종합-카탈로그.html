설치 방법 총정리

스킬 설치는 크게 4가지 방식이 있다.

방식 명령어 예시
Plugin Marketplace /plugin marketplace add <owner>/<repo> → /plugin install <skill>@<marketplace>
npx skills CLI npx skills add <owner>/<repo> --skill <skill-name>
Git Clone (수동) git clone <repo> → 스킬 폴더를 ~/.claude/skills/에 복사
Antigravity (대량설치) npx antigravity-awesome-skills --claude (1,272+ 스킬 일괄 설치)

한국 커뮤니티 핵심 리소스

한국 개발자 커뮤니티에서도 Claude Code 스킬에 대한 활발한 논의와 제작이 이루어지고 있다. 긱뉴스(news.hada.io)에서는 Superpowers, /recall 스킬, tutor-skills, LangChain 스킬 등이 깊이 있게 논의되었고, Velog은 한국어 Claude Code 스킬 콘텐츠의 주요 플랫폼이 되었다. 한국 개발자가 직접 제작한 스킬로는 fireauto(imgompanda), oh-my-claudecode(Yeachan-Heo), claude-skills(jeongsk), claude-integration(m16khb), my-skills(bear2u), tutor-skills(RoundTable02) 등이 있다. 특히 fireauto는 SEO, 보안, PRD, 리서치를 통합한 다목적 플러그인으로, oh-my-claudecode는 "Claude Code를 배울 필요 없이 바로 사용"할 수 있는 멀티에이전트 오케스트레이션 도구로 주목받았다.

결론: 생태계의 현재와 실전 조언

Claude Code 스킬 생태계는 5개월 만에 수만 개 규모로 성장했지만, 실전 테스트에서 의미 있는 품질 향상을 보인 스킬은 전체의 10-20% 수준이라는 것이 여러 커뮤니티 리뷰의 공통 결론이다. 스킬은 12개 이하로 유지하는 것이 정확도에 유리하다(LangChain 사례). Anthropic 공식 스킬(frontend-design, simplify, docx/pdf/pptx/xlsx)과 검증된 대형 프레임워크(Superpowers, ui-ux-pro-max)를 기반으로 시작한 후, 도메인별로 필요한 스킬을 점진적으로 추가하는 전략이 가장 효과적이다. 하드웨어(FPGA/Verilog/VHDL) 분야는 아직 스킬이 부족하여 kicad-happy, circuit-synth 외에는 직접 스킬을 작성해야 할 수 있다. 스킬 활성화 신뢰도 문제는 forced-eval 훅으로 50%에서 80-84%까지 개선 가능하다는 한국 커뮤니티의 실험 결과도 참고할 만하다.
